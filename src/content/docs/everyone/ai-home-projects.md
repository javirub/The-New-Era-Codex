# AI for Home Projects

## Project Planning

```
Prompt: Plan home renovation:

Project: [Kitchen remodel/Bathroom/Deck]
Budget: $[amount]
Timeline: [Months]
DIY level: [Beginner/Intermediate/Expert]
Space: [Dimensions]

Provide:
- Phase breakdown
- Materials list
- Cost estimate
- Timeline
- When to hire pros
```

## DIY Guides

```
Prompt: Step-by-step guide:

Task: [Install ceiling fan/Paint room/Fix leak]
Experience: [My level]
Tools available: [List]

Include:
- Safety precautions
- Tools needed
- Materials with quantities
- Detailed steps
- Common mistakes
- Troubleshooting
```

## Material Selection

```
Prompt: Help me choose:

Project: [Flooring/Paint/Fixtures]
Room: [Description]
Style: [Modern/Traditional/etc]
Budget: [Per sq ft]
Usage: [High/Low traffic]

Recommend options with pros/cons.
```

## Cost Estimation

```
Prompt: Estimate costs:

Project: [Description]
Location: [City, State]
Size: [Square feet/units]
Quality: [Budget/Mid/High-end]

Break down:
- Materials
- Labor (if hiring)
- Permits
- Contingency
- Total range
```

## Before & After Vision

```
Prompt: Describe transformation:

Current: [Room description]
Goals: [What I want]
Budget: [Amount]
Style: [Preferences]

Suggest:
- Quick wins
- Major changes
- Phased approach
- Expected outcome
```

## Troubleshooting

```
Prompt: Diagnose problem:

Issue: [Leaky faucet/Flickering light/etc]
Observations: [What you see/hear]
When it happens: [Timing]
What you've tried: [Attempts]

Provide:
- Likely causes
- DIY fixes
- When to call professional
- Safety concerns
```
