# AI for Personal Finance

## Budget Creation

```
Prompt: Create monthly budget:

Income: $[amount]
Fixed expenses:
- Rent: $[amount]
- Utilities: $[amount]
- Insurance: $[amount]

Financial goals:
- Emergency fund: $[target]
- Retirement: [%]
- Savings: [purpose]

Recommend allocation using 50/30/20 rule.
```

## Expense Analysis

```
Prompt: Analyze my spending:

Last month expenses:
[Paste list or summary]

Identify:
- Unnecessary spending
- Opportunities to save
- Spending patterns
- Budget optimization
- Action items
```

## Savings Goals

```
Prompt: Create savings plan:

Goal: $[amount] for [purpose]
Timeline: [months/years]
Current savings: $[amount]
Monthly budget: $[amount]

Provide:
- Monthly savings needed
- Investment strategy
- Milestones
- Motivation tips
```

## Debt Payoff Strategy

```
Prompt: Plan debt payoff:

Debts:
1. Credit card: $[amount] at [%] APR
2. Student loan: $[amount] at [%] APR
3. Car loan: $[amount] at [%] APR

Monthly available: $[amount]

Compare:
- Avalanche method (highest rate first)
- Snowball method (smallest first)

Recommend best approach.
```

## Investment Research

```
Prompt: Explain investment option:

Investment: [ETF/Stock/Fund name]
My profile:
- Age: [age]
- Risk tolerance: [low/medium/high]
- Goals: [retirement/growth/income]
- Timeline: [years]

Explain:
- What it is
- Pros/cons
- Fit for my profile
- Risks
- Alternatives

⚠️ Not financial advice - do your research!
```

## Financial Education

```
Prompt: Explain like I'm 5:

Concept: [401k/IRA/compound interest/etc]

Include:
- Simple explanation
- Why it matters
- How to get started
- Common mistakes
- Resources to learn more
```

## Tax Planning

```
Prompt: Tax optimization ideas:

Income: $[amount]
Filing: [Single/Married]
State: [State]
Deductions: [list current]

Suggest:
- Often-missed deductions
- Tax-advantaged accounts
- Timing strategies
- Documentation needs

⚠️ Consult a tax professional!
```

## Bill Negotiation

```
Prompt: Help me negotiate this bill:
```

## Important Disclaimers

⚠️ **AI is NOT**:
- A licensed financial advisor
- Able to see your full situation
- Responsible for outcomes
- A replacement for professionals

✅ **AI CAN**:
- Help organize thoughts
- Explain concepts
- Generate ideas
- Provide general education
- Create templates

**Always**:
- Verify information
- Consult professionals for major decisions
- Consider your unique situation
- Use multiple sources
- Update information regularly
