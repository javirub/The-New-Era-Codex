# 50 Magic Prompt Templates: Copy, Paste, Get Results

## How to Use This Guide

Each template has:
- **Purpose**: What it's for
- **Template**: Copy-paste ready (replace parts in [brackets])
- **Example**: See it in action

**Pro tip**: Save your favorites and customize them for your needs.

## Writing & Communication (10 Templates)

### 1. Professional Email Writer

**Purpose**: Write any business email quickly

**Template**:
```
Write a professional email to [recipient] about [topic].
Tone: [formal/friendly/apologetic]
Key points to include:
- [point 1]
- [point 2]
- [point 3]
Keep it under [word count] words.
```

**Example**:
```
Write a professional email to my team about the project deadline extension.
Tone: friendly but clear
Key points to include:
- New deadline is March 15
- Reasons for extension (client feedback delays)
- What we'll accomplish with extra time
Keep it under 150 words.
```

### 2. Text Improver

**Purpose**: Make any text better

**Template**:

## Pro Tips for All Templates

### 1. Layer Your Prompts

Start broad, then refine:
```
First: [Use template]
Then: "Make it more formal"
Then: "Add a section about X"
Then: "Shorten to 200 words"
```

### 2. Add Examples

```
[Template] +

Like this example: [paste example]
Match the style but change the content.
```

### 3. Specify Format

```
[Template] +

Format output as:
- Bullet points
- Numbered list
- Table
- Markdown
```

### 4. Request Variations

```
[Template] +

Give me 3 different versions:
1. Formal
2. Casual
3. Creative
```

### 5. Iterate

Don't accept first output if it's not perfect:
```
"Good, but can you make it [more specific request]"
```
