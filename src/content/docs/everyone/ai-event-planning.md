# AI for Event Planning

## Event Conceptualization

```
Prompt: Plan [event type]:

Event: [Birthday/Wedding/Conference]
Guests: [Number]
Budget: $[amount]
Date: [Date]
Venue: [Type/Location]
Theme: [If any]

Provide:
- Theme ideas
- Venue suggestions
- Budget breakdown
- Timeline
- Vendor types needed
```

## Guest List Management

```
Prompt: Help organize guest list:

Event: [Type]
Total capacity: [Number]
Categories: [Family/Friends/Work/etc]

Create:
- Invitation priority
- Seating arrangement ideas
- Dietary tracking system
- RSVP management plan
```

## Menu Planning

```
Prompt: Create menu:

Event: [Type]
Guests: [Number]
Dietary needs: [List]
Style: [Buffet/Plated/Cocktail]
Budget: $[per person]
Season: [Season]

Suggest:
- Appetizers
- Main courses
- Desserts
- Beverages
- Quantities
```

## Timeline & Checklist

```
Prompt: Event planning timeline:

Event: [Type]
Date: [Date]
Complexity: [Simple/Moderate/Complex]

Create checklist:
- 6 months before
- 3 months before
- 1 month before
- 1 week before
- Day before
- Day of
```

## Vendor Research

```
Prompt: Find vendors for:

Event: [Type]
Location: [City]
Budget: [Range]
Style: [Preferences]

List vendor types needed:
- What they provide
- Questions to ask
- Typical costs
- Booking timeline
```

## Decoration Ideas

```
Prompt: Decoration plan:

Event: [Type]
Theme: [Theme]
Venue: [Description]
Budget: $[amount]
DIY willing: [Yes/No]

Suggest:
- Color scheme
- Centerpieces
- Lighting
- Entry decoration
- Photo areas
- DIY vs buy recommendations
```

## Invitation Wording

```
Prompt: Write invitation:

Event: [Type]
Formality: [Casual/Formal]
Key details:
- Date/time
- Location
- Dress code
- RSVP info

Include appropriate wording and etiquette.
```

## Day-of Schedule

```
Prompt: Create day-of timeline:

Event: [Wedding/Party]
Start: [Time]
Duration: [Hours]
Key moments: [List]

Create:
```
