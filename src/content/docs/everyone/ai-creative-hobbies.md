# AI for Creative Hobbies

## Creative Writing

### Story Ideas
```
Prompt: Generate story ideas:

Genre: [Mystery/Romance/Sci-Fi]
Themes: [List]
Length: [Short story/Novel]
Unique twist: [Optional constraint]

Give me 10 concepts.
```

### Character Development
```
Prompt: Develop a character:

Role: [Protagonist/Antagonist]
Genre: [Genre]
Archetype: [Starting point]

Create:
- Background
- Personality traits
- Motivations
- Flaws
- Character arc
- Dialogue style
```

### Writing Feedback
```
Prompt: Critique this passage:

[Paste your writing]

Feedback on:
- Pacing
- Dialogue naturalness
- Show vs tell
- Descriptions
- Emotional impact

Be constructive.
```

## Visual Art

### Composition Ideas
```
Prompt: Suggest compositions for:

Subject: [Landscape/Portrait/Still life]
Style: [Realistic/Abstract/Impressionist]
Mood: [Calm/Dramatic/Whimsical]

Describe 5 interesting compositions.
```

### Color Palettes
```
Prompt: Create color palette:

Theme: [Sunset/Ocean/Autumn]
Colors: [Number]
Style: [Warm/Cool/Complementary]

Provide hex codes and usage suggestions.
```

## Music

### Song Structure
```
Prompt: Help me structure a song:

Genre: [Pop/Rock/Jazz]
Theme: [Love/Loss/Adventure]
Length: [Minutes]

Suggest:
- Verse/chorus structure
- Tempo
- Key
- Instrumentation
- Lyric themes per section
```

### Lyrics Writing
```
Prompt: Write song lyrics:

Theme: [Topic]
Style: [Artist/Genre]
Mood: [Happy/Melancholic]
Length: [3-4 minutes]

Include:
- Verses
- Chorus
- Bridge
- Rhyme scheme
```

## Photography

### Shot Planning
```
Prompt: Plan photo shoot:

Subject: [Portrait/Landscape/Product]
Location: [Description]
Time: [Golden hour/Midday/Blue hour]
Equipment: [What you have]

Suggest:
- Shot list
- Angles
- Lighting setup
- Camera settings
- Composition tips
```

### Photo Editing Ideas
```
Prompt: Editing suggestions:

Photo description: [What's in frame]
Current feel: [Current mood]
Desired feel: [Target mood]
Style: [Moody/Bright/Vintage]

Suggest adjustments to:
- Exposure
- Colors
- Contrast
- Specific edits
```

## Crafts & Making

### Project Ideas
```
Prompt: Craft project ideas:

Skill: [Knitting/Woodworking/etc]
Level: [Beginner/Advanced]
Materials: [What you have]
Time: [Available hours]

Suggest 5 projects with difficulty ratings.
```

## Skill Development
