# AI for Language Learning

## Conversation Practice

```
Prompt: Let's have a conversation in [Spanish]:

My level: [Beginner/Intermediate/Advanced]
Topic: [Daily life/Travel/Business]

Rules:
- Speak at my level
- Correct my mistakes in [brackets]
- Explain grammar when I ask
- Mix formal/informal appropriately
```

## Grammar Explanation

```
Prompt: Explain [grammar concept] in [language]:

Use:
- Simple terms
- Examples
- When to use it
- Common mistakes
- Practice sentences
```

## Translation & Context

```
Prompt: Translate and explain:

Phrase: [phrase in target language]

Provide:
- Literal translation
- Natural translation
- When to use it
- Cultural context
- Similar expressions
```

## Vocabulary Building

```
Prompt: Teach me 20 [theme] words in [language]:

Theme: [Food/Travel/Business]
Level: [Your level]

For each:
- Word
- Pronunciation
- Example sentence
- Memory tip
```

## Writing Practice

```
Prompt: Review my [language] writing:

My text: [paste]

Check:
- Grammar
- Vocabulary
- Natural flow
- Cultural appropriateness

Provide corrections with explanations.
```

## Tools

- **Duolingo**: AI-powered lessons
- **ChatGPT**: Conversation partner
- **ElevenLabs**: Pronunciation practice
