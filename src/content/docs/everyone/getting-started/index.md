# Getting Started with AI

New to AI? Start here! These guides will teach you the basics in simple, jargon-free language. No technical background required.

<Aside type="tip" title="Perfect for Beginners">
These guides assume zero prior knowledge. If you can use a web browser, you can follow these tutorials.
</Aside>

## What You'll Learn

By completing these getting-started guides, you'll be able to:
- ✅ Use ChatGPT confidently for everyday tasks

## Common Questions

**Do I need to be technical?**
No! These guides are written for everyone, regardless of technical background.

**Is it free?**
ChatGPT has a free tier. Some advanced features require a subscription, but you can learn the basics for free.

**How long will it take?**
Each guide takes 10-20 minutes. You can complete all four in about an hour.

**What if I get stuck?**
Every guide includes troubleshooting tips and common mistakes to avoid.
