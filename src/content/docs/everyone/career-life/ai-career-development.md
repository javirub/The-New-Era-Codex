# AI for Career Development

## Resume & Cover Letter

**Resume optimization**:
```
Review my resume for a [job title] position:

[paste resume]

Target job description:
[paste job description]

Provide:
- ATS keyword matches
- Accomplishments to highlight
- Weaknesses to address
- Formatting improvements
- Bullet point rewrites (action verbs, quantifiable results)
```

**Accomplishment framing**:
```
Help me rewrite these work experiences using the "STAR" method (Situation, Task, Action, Result):

Current: "Managed social media accounts"

Make it: Quantifiable, impressive, specific, achievement-focused
```

**Cover letter**:
```
Write a cover letter for:
- Position: [job title] at [company]
- Why I'm interested: [your reasons]
- Key qualifications: [your experience]
- What I bring: [unique value]

Tone: Professional but personable
Length: 3 paragraphs
```

## Interview Preparation

**Common questions**:
```
I have an interview for [position]. Prepare me for common questions:

1. "Tell me about yourself"
2. "Why do you want to work here?"
3. "What's your greatest weakness?"
4. "Where do you see yourself in 5 years?"

For each, give me:
- Framework for answering
- Sample answer tailored to my background
- What to avoid
```

**STAR method practice**:
```
Help me prepare STAR stories for:
- A time I showed leadership
- A conflict I resolved
- A failure and what I learned
- A time I went above and beyond

For each, ask me questions to draw out details, then help me structure the story compellingly.
```

**Behavioral questions**:
```
Generate 20 behavioral interview questions for a [job type] role, organized by theme:
- Leadership
- Problem-solving
- Teamwork
- Adaptability
- Results/achievement
```

**Questions to ask interviewer**:
```
I'm interviewing for [position] at [company]. Suggest 10 thoughtful questions I should ask:

- About the role
- About team culture
- About growth opportunities
- About company direction

That show I'm: Engaged, strategic, culture-conscious
```

## Career Planning

**Skills gap analysis**:
```
I'm currently a [current role] and want to become a [goal role] in 2 years.

Compare:
- Skills I have
- Skills I need
- How to acquire missing skills
- Timeline for each
- Certifications worth getting
```

**Career roadmap**:
```
Create a 5-year career plan:

Starting point: [current role/level]
Goal: [dream role/level]
Industry: [your industry]
Constraints: [e.g., can't relocate, need to keep current salary]

Include: Milestones, skill development, networking goals, possible pivots
```

## Professional Branding

**LinkedIn profile**:

```
Optimize my LinkedIn headline and summary:

Current: [paste]
Role: [job title]
Industry: [industry]
What makes me unique: [your differentiators]

Make it: Searchable, compelling, authentic
```

**Personal brand statement**:
```
Help me craft a personal brand statement:
- What I do: [role/expertise]
- Who I serve: [target audience]
- How I add value: [your unique approach]
- Career goals: [where you're heading]

Keep it: One sentence, memorable, authentic
```

## Networking

**Outreach messages**:
```
Write a LinkedIn message to:
- Who: Someone at my dream company
- Purpose: Informational interview
- Connection: [how you found them]
- What you want: 20-min call to learn about their role

Tone: Professional, respectful of their time, specific ask
```

**Elevator pitch**:
```
Create a 30-second elevator pitch:
- Who I am: [background]
- What I do: [current role]
- What I'm looking for: [opportunities/connections]
- Memorable hook: [interesting fact or achievement]

3 versions: Casual networking, formal event, job fair
```

**Follow-up after networking**:
```
I met [person] at [event] and we discussed [topic]. Write a follow-up email that:
- Thanks them
- References our conversation
- Provides value (article/connection/insight)
- Suggests next step
- Isn't pushy
```

## Salary Negotiation

**Research & preparation**:
```
I'm negotiating for a [job title] role in [city/industry].

Help me:
- Research fair salary range
- Calculate my target (based on experience, skills, market)
- Prepare negotiation talking points
- Plan for different scenarios (low offer, non-negotiable, benefits instead)
```

**Negotiation scripts**:
```
Create response scripts for:

Scenario 1: Offer is lower than expected
Scenario 2: They ask my salary expectations
Scenario 3: Negotiating benefits if salary is fixed
Scenario 4: Accepting the offer professionally

For each: What to say, what not to say, how to maintain relationship
```

## Performance Reviews

**Self-assessment**:
```
My annual review is coming up. Help me prepare:

Achievements this year:
- [list]

Challenges I overcame:
- [list]

Create:
- Compelling summary of my year
- Quantifiable impact statements
- Growth areas (framed positively)
- Goals for next year
```

**Asking for a raise**:
```
I want to ask for a raise. Help me build my case:

- Time since last raise: [time]
- New responsibilities: [list]
- Achievements: [list]
- Market rate comparison: [research]

Create:
- Opening statement
- Evidence to present
- Amount to request
- How to handle pushback
```

## Skill Development

**Learning plan**:
```
I need to learn [skill] for career advancement.

Create a 90-day plan:
- What to learn in which order
- Best resources (free and paid)
- Practice projects
- How to demonstrate this skill
- How to add to resume/portfolio
```

**Certification roadmap**:
```
I'm in [field]. Which certifications would most benefit my career?

Consider:
- ROI (time and money vs. value)
- Industry recognition
- My career goals
- Current market demand

Prioritize and create timeline.
```

## Professional Writing

**Email etiquette**:
```
Help me write a professional email to:
- Recipient: [Senior executive/client/colleague]
- Purpose: [request/update/apology]
- Tone: [formal/friendly-professional]
- Length: Keep concise

Template: Greeting, context, ask/message, close
```

**Business proposals**:
```
Outline a business proposal for:
- What: [your idea/project]
- Problem it solves: [issue]
- Solution: [approach]
- Benefits: [outcomes]
- Resources needed: [budget/time/people]

Structure: Executive summary, problem, solution, implementation, budget
```

## Leadership Development

**Management scenarios**:
```
I'm a new manager facing this situation:

[describe challenge with team member/project/conflict]

What should I:
1. Do immediately
2. Avoid doing
3. Say (specific phrases)
4. Learn from this for future
```

**Delegation framework**:
```
I struggle with delegating. Create a framework for deciding:
- What tasks to delegate
- To whom
- How much guidance to give
- How to follow up without micromanaging
- When to step in vs. let them learn
```

```
I'm considering moving into [industry]. Tell me:
```

## Work-Life Balance

**Time management**:
```
I'm overwhelmed with work. Help me:
- Prioritize my tasks: [list everything]
- Identify what to delegate/eliminate
- Create a realistic schedule
- Set boundaries
- Say no professionally
```

**Burnout prevention**:
```
I'm feeling burnt out. Create a plan to:
- Assess what's causing it
- Set healthier boundaries
- Communicate needs to manager
- Build sustainable work habits
- Know when it's time to change jobs
```

## Side Hustles & Freelancing

**Freelance setup**:
```
I want to start freelancing as a [skill/service].

Help me:
- Price my services
- Create a simple portfolio
- Write a service description
- Find first clients
- Manage client communication
```

**Side hustle ideas**:
```
Given my skills: [list]
And constraints: [time available, resources]

Suggest 5 side hustle ideas that:
- Can start with minimal investment
- Have actual market demand
- Fit my schedule
- Could scale if successful
```
