# AI for Job Search

## Resume Optimization

### Tailoring to Job Description

```
Prompt: Optimize my resume for this job:

My resume: [paste resume]

Job description: [paste JD]

Tailor my:
- Summary
- Skills section
- Experience descriptions
- Keywords for ATS

Maintain truthfulness.
```

### ATS-Friendly Formatting

```
Prompt: Review my resume for ATS compatibility:

[paste resume]

Check for:
- Parsing issues
- Keyword optimization
- Format problems
- Missing sections
```

## Cover Letter Generation

```
Prompt: Write a cover letter:

Job: [title] at [company]
Key requirements: [from JD]
My relevant experience:
- [Experience 1]
- [Experience 2]
- [Experience 3]

Tone: Professional but personable
Length: 250-300 words
Highlight: why I'm a great fit
```

## Interview Preparation

### Company Research
```
Prompt: Research brief for interview:

Company: [Company name]
Industry: [Industry]
Role: [Position]

Provide:
- Company overview
- Recent news
- Competitors
- Culture insights
- Questions to ask
```

### Common Questions
```
Prompt: Help me prepare answers for:

1. "Tell me about yourself"
2. "Why this role?"
3. "Greatest strength/weakness"
4. "Where do you see yourself in 5 years?"

Based on:
- Role: [Job title]
- My background: [Summary]
- Company: [Company name]

Use STAR method for examples.
```

### Mock Interview
```
Prompt: Conduct mock interview:

Role: [Position]
Industry: [Industry]
Experience: [My level]

Ask me 5 interview questions one at a time.
After each answer, provide feedback on:
- Strengths
- Areas to improve
- Suggested revisions
```

## Networking Messages

### LinkedIn Connection
```
Prompt: Write LinkedIn connection message:

To: [Name], [Title] at [Company]
Context: [How you found them]
Common ground: [Shared interest/experience]
Goal: [What you hope for]

Length: Under 300 characters
Tone: Professional, genuine
```

### Informational Interview
```
Prompt: Request informational interview:

To: [Name]
About: [Their role/company]
Why: [Your interest]
Ask: 15-20 minute call

Make it easy to say yes.
```

### Thank You Email
```

- [Point 1]
- [Point 2]

Include:
- Gratitude
- Reiterate interest
- Reference specific conversation
- Next steps

```

### Follow-Up
```
Prompt: Write follow-up email:

Situation: Applied [X] weeks ago, no response
Company: [Company name]
Position: [Role]

Goal: Gentle reminder
Tone: Professional, not pushy
Length: Under 100 words
```

## Best Practices

✅ **Do**:
- Personalize AI output
- Fact-check everything
- Maintain authenticity
- Use as starting point
- Practice answers out loud

❌ **Don't**:
- Copy-paste without editing
- Fabricate experience
- Use generic applications
- Rely solely on AI
- Skip proofreading
