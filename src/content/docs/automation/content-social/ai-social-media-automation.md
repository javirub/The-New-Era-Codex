# AI Social Media Automation

## Content Generation Workflow

```
1. Generate post ideas (ChatGPT)
2. Create visuals (DALL-E/Canva)
3. Optimize captions (AI tools)
4. Schedule posts (Buffer/Hootsuite)
5. Monitor engagement (AI analytics)
6. Respond automatically (chatbots)
```

## Tools Stack

**Content Creation**:
- ChatGPT: Post copywriting
- DALL-E/Midjourney: Images
- Canva AI: Design templates
- Lately: Repurpose long-form content

**Scheduling**:
- Buffer: Multi-platform scheduling
- Hootsuite: Enterprise management
- Later: Visual planning

**Analytics**:
- Sprout Social: AI-powered insights
- Brandwatch: Social listening

## Prompt Examples

**Post generation**:
```
Create 10 LinkedIn posts about [topic]:
- Hook-driven opens
- Value-focused content
- Clear CTAs
- Mix: tips, stories, questions
- 150-200 words each
```

**Hashtag research**:
```
Suggest hashtags for [content]:
- 3 high-traffic hashtags
- 5 niche-specific hashtags
- 2 branded hashtags
Explain strategy for each
```

**Engagement responses**:
```
Draft responses to common comments:
1. "Tell me more"
2. "Thanks for sharing"
3. "I disagree because..."

Maintain brand voice: [description]
```

## Automation Workflow (Zapier)

```
1. Trigger: RSS feed new post
2. AI: Generate social post from article
3. AI: Create image prompt
4. DALL-E: Generate image
5. Buffer: Schedule post
6. Slack: Notify team
```
