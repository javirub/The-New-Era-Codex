**Lo que quieres**: [ej., "Ayuda para escribir un correo profesional"]

```

```

**Por qué es malo**: Demasiado vago - obtendrás algo genérico que no puedes usar realmente.

```

mueve a las 2pm. Mantenlo amigable pero breve. Menos de 100 palabras.
```

**Por qué es bueno**: Destinatario específico, mensaje claro, tono especificado, longitud limitada.
