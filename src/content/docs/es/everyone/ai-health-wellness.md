# IA para Salud y Bienestar

## Planificación de Fitness

### Generación de Plan de Ejercicios
```
Prompt: Crea plan de ejercicios:

Objetivo: [Pérdida de peso/Ganancia muscular/Fitness general]
Experiencia: [Principiante/Intermedio/Avanzado]
Disponible: [Equipo]
Tiempo: [Minutos por sesión]
Frecuencia: [Días por semana]
Restricciones: [Lesiones/limitaciones]

Proporciona:
- Horario semanal
- Ejercicios (series/repeticiones)
- Plan de progresión
- Días de descanso
```

### Descripciones de Verificación de Forma
```
Prompt: Describe la forma adecuada:

Ejercicio: [Nombre del ejercicio]
Errores comunes a evitar
Puntos clave en los que enfocarse
Modificaciones para principiantes
```

## Nutrición y Planificación de Comidas

### Planes de Comidas Personalizados
```
Prompt: Crea plan de comidas:

Objetivo: [Pérdida de peso/Ganancia/Mantenimiento]
Dietético: [Restricciones/preferencias]
Calorías: [Objetivo]
Comidas por día: [Número]
Tiempo de cocina: [Máximo minutos]
Presupuesto: [Por día]

Incluye:
- Desayuno/almuerzo/cena/snacks
- Recetas
- Lista de compras
- Desglose de macros
```

### Modificaciones de Recetas
```
Prompt: Modifica esta receta:

Original: [Receta]

Hazla:
- Baja en calorías
- Alta en proteína
- Sin lácteos
- Vegetariana

Mantén sabor similar.
```

## Seguimiento y Análisis de Salud

### Verificador de Síntomas
```
Prompt: Ayúdame a entender síntomas:

Síntomas: [Lista]
Duración: [Tiempo]
Severidad: [1-10]

Proporciona:
- Causas posibles
- Cuándo ver al médico
- Remedios caseros
- Preguntas para el médico

⚠️ NO es consejo médico - ¡consulta a un profesional!
```

### Recordatorios de Medicamentos
```
Crear sistema:
- Horarios para tomar
- Con/sin alimentos
- Efectos secundarios a vigilar
- Interacciones a evitar
```

## Descargos de Responsabilidad Importantes

⚠️ **Crítico**: La IA NO es:
- Un médico
- Un nutricionista licenciado
- Un terapeuta
- Un reemplazo para atención médica

✅ **La IA PUEDE**:
- Proporcionar información general
- Ayudar a organizar pensamientos
- Sugerir enfoques comunes
- Rastrear progreso
- Motivar y apoyar

**Siempre**:
- Consultar profesionales de salud
- Verificar información médica
- Revelar a tu médico
- Emergencia = llamar al 911
- Crisis mental = llamar a línea de ayuda
