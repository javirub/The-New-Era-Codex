# IA como tu Asistente Personal

## Planificación Diaria con IA

**Prompt de rutina matutina**:
```
Necesito planificar mi día. Esto es lo que tengo:

Agenda:
- 9am: Reunión de equipo
- 2pm: Cita médica
- 4pm: Recoger niños

Tareas:
- Terminar propuesta de proyecto
- Comprar víveres
- Llamar al seguro

Prioridades: La propuesta es urgente, el seguro puede esperar

Crea un cronograma optimizado con bloques de tiempo y sugiere cuándo hacer cada tarea.
```

**La IA te ayudará a**:
- Priorizar tareas
- Encontrar bloques de tiempo
- Considerar tiempo de viaje
- Sugerir necesidades de preparación

## Planificación de Comidas y Recetas

**Planificación semanal de comidas**:
```
Planifica 7 cenas saludables para mi familia:
- 2 adultos, 2 niños (edades 6 y 9)
- Sin mariscos (alergia)
- Presupuesto: $150 para la semana
- Tiempo de preparación: máximo 30 minutos
- Usar vegetales de temporada

Proporciona:
1. Menú para cada día
2. Lista completa de compras
3. Recetas rápidas
```

**Modificación de recetas**:
```
Tengo pollo, arroz y brócoli. Crea una receta de cena sabrosa que tome menos de 30 minutos.
```

## Planificación de Viajes

**Creación de itinerario**:
```
Planifica un viaje de 3 días a Barcelona:
- Presupuesto: $1000 (excluyendo vuelos)
- Intereses: Historia, comida, arquitectura
- Ritmo: Relajado, sin prisas
- Imprescindible: Sagrada Familia

Crea itinerario día a día con:
- Actividades y horarios
- Sugerencias de restaurantes
- Transporte entre lugares
- Desglose de presupuesto
```

**Generador de lista de equipaje**:
```
Voy a Islandia en noviembre por 5 días. ¿Qué debería empacar?

Actividades planeadas:
- Senderismo
- Tour de auroras boreales
- Exploración de la ciudad
```

## Gestión de Correo Electrónico

**Redacción de correo**:
```
Escribe un correo electrónico cortés a mi arrendador solicitando:
- Arreglar la llave del baño que gotea
- Revisar el sistema de calefacción (no funciona correctamente)
- Programar un horario que funcione para ambos

Mantenlo profesional pero amigable.
```

**Respuestas de correo**:
```
Responde a este correo profesionalmente pero declina cortésmente:

[pega invitación de correo que quieres declinar]

Razón: Conflicto de horario
Tono: Cálido pero firme
```

## Ideas de Regalos

**Sugerencias personalizadas**:
```
Necesito ideas de regalo para el cumpleaños de mi esposa:
- Edad: 35
- Intereses: Yoga, lectura, cocina
- Presupuesto: $100-$200
- Algo único, no genérico

Sugiere 5 opciones reflexivas con razones de por qué le gustaría cada una.
```

## Gestión del Hogar

**Cronograma de mantenimiento del hogar**:
```
Crea una lista de verificación de mantenimiento del hogar para mí:
- Casa: 3 dormitorios, 2 baños
- Tiene: Jardín, sistema de calefacción, calentador de agua
- Ubicación: Clima frío

¿Qué debería revisar mensual, trimestral y anualmente?
```

**Rutina de limpieza**:
```
Diseña un cronograma de limpieza realista:
- Trabajo tiempo completo
- 2 niños
- Casa de 1,500 pies cuadrados
- Quiero mantener la casa ordenada pero no pasar horas diarias

Divídelo por: tareas diarias, semanales, mensuales
```

## Planificación Financiera

**Asistente de presupuesto**:
```
Ayúdame a crear un presupuesto mensual:

Ingresos: $5,000/mes

Gastos fijos:
- Alquiler: $1,500
- Pago del auto: $400
- Seguro: $200
- Servicios: $150

Variable:
- Comida: ?
- Entretenimiento: ?
- Meta de ahorro: 20%

¿Qué es razonable para las categorías variables?
```

**Metas de ahorro**:
```
Quiero ahorrar $10,000 en 18 meses para vacaciones. Actualmente puedo ahorrar $300/mes.

- ¿Estoy en camino?
- ¿Qué pasa si reduzco comer fuera en $100/mes?
- Muéstrame las matemáticas y un plan mensual
```

## Actividades y Educación de Niños

**Ayuda con tareas**:
```
Explica la fotosíntesis a un niño de 9 años de manera divertida y simple. Incluye una analogía con la que puedan relacionarse.
```

**Ideas de actividades**:
```
Mis niños (edades 6 y 9) están aburridos un sábado lluvioso. Sugiere 5 actividades de interior:
- Bajo/sin costo
- Educativas pero divertidas
- 30-60 minutos cada una
- Se pueden hacer con artículos de casa
```

## Salud y Fitness

**Planes de ejercicio**:
```
Crea un plan de fitness de 4 semanas para principiantes:
- Meta: Fitness general y pérdida de peso
- Tiempo: 30 minutos, 4 días/semana
- Equipo: Ninguno (solo peso corporal)
- Nivel actual: Sedentario

Incluye calentamiento, ejercicio, enfriamiento para cada día.
```

**Ayuda con seguimiento nutricional**:
```
Comí esto hoy - ¿es saludable?

Desayuno: Café, bagel con queso crema
Almuerzo: Ensalada César de pollo
Cena: Pasta con marinara
Snacks: Manzana, barra de granola

¿Qué está bien, qué podría mejorar?
```

## Aprendizaje y Desarrollo de Habilidades

**Planes de estudio**:
```
Quiero aprender Python básico en 3 meses, estudiando 30 minutos diarios.

Crea un plan de aprendizaje estructurado con:
- Temas semanales
- Microlecciones diarias
- Ejercicios de práctica
- Proyectos hito
```

**Aprendizaje de idiomas**:
```
Enséñame 10 frases esenciales en español para mi viaje el próximo mes. Incluye:
- Guía de pronunciación
- Cuándo usar cada frase
- Consejos culturales
```
