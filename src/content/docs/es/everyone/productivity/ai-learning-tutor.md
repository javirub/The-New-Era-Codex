# IA como tu Tutor Personal

## La Técnica Feynman

**Aprender enseñando**:
```
Explícame [física cuántica/aprendizaje automático/economía] como si tuviera 10 años.

Empieza con lo absolutamente básico, usa analogías de la vida cotidiana y aumenta la complejidad lentamente.
```

**Por qué funciona**: Si la IA puede explicarlo de forma simple, lo entenderás profundamente.

## Rutas de Aprendizaje Personalizadas

**Hoja de ruta de habilidades**:
```
Quiero aprender [desarrollo web/fotografía/español] desde cero.

Crea una ruta de aprendizaje de 90 días con:
- Progresión semana por semana
- Tareas diarias específicas (30-60 min)
- Proyectos hito
- Recursos para cada etapa
- Cómo saber cuándo estoy listo para avanzar

Nivel actual: Principiante completo
Estilo de aprendizaje: Práctico, ejemplos prácticos
```

## Desglosando Temas Complejos

**ELI5 (Explícalo Como si Tuviera 5 Años)**:
```
Estoy leyendo sobre blockchain pero no lo entiendo. Explícalo usando:
- Una analogía simple
- Proceso paso a paso
- Por qué importa
- Conceptos erróneos comunes a evitar
```

**Complejidad progresiva**:
```
Explica la fotosíntesis en 3 versiones:
1. Para un niño de 5 años
2. Para un estudiante de secundaria
3. Para un estudiante universitario de biología

Te diré qué nivel entiendo, y construiremos desde ahí.
```

## Recuerdo Activo y Cuestionarios

**Preguntas de práctica**:
```
Estoy estudiando [Segunda Guerra Mundial/cálculo/programación en Python].

Crea 10 preguntas de prueba que:
- Prueben comprensión real, no solo memorización
- Vayan de básico a avanzado
- Incluyan explicaciones de respuestas correctas
- Identifiquen errores comunes
```

**Ayudante de repetición espaciada**:
```
Estoy aprendiendo vocabulario en español. Aquí hay 20 palabras que estudié esta semana:
[lista de palabras]

Crea un cronograma de repaso:
- Cuándo repasar (hoy, 2 días, 1 semana, 1 mes)
- Pruébame con diferentes tipos de preguntas
- Mezcla palabras antiguas que aún debería recordar
```

## Técnicas de Estudio

**Resumen de notas**:
```
Aquí están mis notas de clase de la conferencia de hoy sobre [tema]:

[pega notas]

Ayúdame a crear:
1. Un resumen de una página de conceptos clave
2. 5 conclusiones importantes
3. Conexiones con material aprendido previamente
4. Posibles preguntas de examen
```

**Mapeo de conceptos**:
```
Crea un mapa de conceptos para [tema] mostrando:
- Concepto principal en el centro
- Subtemas clave ramificándose
- Cómo los conceptos se conectan entre sí
- Aplicaciones del mundo real
```

## Aprendizaje de Idiomas

**Práctica de conversación**:
```
Tengamos una conversación en español sobre ordenar comida en un restaurante.

Reglas:
- Tú haces el mesero, yo soy el cliente
- Mantenlo a nivel A2 (principiante)
- Si cometo un error, corrígeme gentilmente
- Haz preguntas de seguimiento para mantenerlo
```

**Aclaración de gramática**:
```
Sigo confundiendo [ser vs. estar / who vs. whom / pasado perfecto vs. pasado simple].

Explica:
- La diferencia real
- Regla simple para recordar cuándo usar cada uno
- 5 oraciones de ejemplo para cada uno
- Un truco para recordarlo
```

## Matemáticas y Ciencia

**Resolución de problemas paso a paso**:
```
Resuelve este problema de cálculo, pero muéstrame cada paso individual:

[problema]

Para cada paso, explica:
- Qué estás haciendo
- Por qué lo estás haciendo
- Qué regla/fórmula estás usando
```

**Problemas de palabras**:
```
Tengo dificultades con problemas de palabras. Aquí hay uno:

[problema]

Ayúdame a:
1. Identificar qué pregunta el problema
2. Qué información se da
3. Qué fórmula/enfoque usar
4. Solución paso a paso
5. Cómo verificar si mi respuesta tiene sentido
```

## Preparación para Exámenes

**Plan de estudio**:
```
Tengo un examen en 2 semanas sobre [materia]. Temas cubiertos:
- [Tema 1]
- [Tema 2]
- [Tema 3]

Crea un cronograma de estudio que:
- Cubra todos los temas
- Incluya recuerdo activo
- Construya hasta exámenes de práctica
- Deje tiempo para repaso
- Considere 2 horas/día de tiempo de estudio
```

**Exámenes de práctica**:
```
Crea un examen de práctica para [materia] que:
- Coincida con el formato de [SAT/AP/final universitario/certificación]
- Pruebe conceptos clave
- Incluya límites de tiempo
- Tenga explicaciones detalladas de respuestas
```

## Habilidades de Investigación

**Exploración de tema**:
```
Necesito escribir un trabajo de investigación sobre [cambio climático/inteligencia artificial/arte renacentista].

Ayúdame a:
1. Reducir a un ángulo específico
2. Desarrollar una declaración de tesis
3. Bosquejar argumentos principales
4. Sugerir tipos de fuentes a buscar
5. Identificar posibles contraargumentos
```

**Evaluación de fuentes**:
```
Encontré esta información sobre [tema]:

[pega contenido]

Ayúdame a evaluar:
- ¿Es esto creíble?
- ¿Qué sesgos podría tener?
- ¿Qué falta de esta perspectiva?
- ¿Qué más debería investigar?
```

## Comprensión de Lectura

**Análisis de texto**:
```
Estoy leyendo [libro/artículo] y tengo dificultades para entender [capítulo/sección].

Aquí hay un pasaje:
[pega texto]

Explica:
- ¿Cuál es la idea principal?
- ¿Qué recursos literarios se están usando?
- ¿Cuál es el propósito del autor?
- ¿Contexto histórico/cultural que debería conocer?
```

**Lectura rápida**:
```
Resume este artículo en 3 viñetas - solo puntos principales:

[pega artículo largo]

Luego hazme preguntas para verificar si entendí las ideas clave.
```

## Técnicas de Memoria

**Mnemotecnia**:
```
Necesito memorizar [orden de planetas/tabla periódica/fechas históricas].

Crea:
- Un acrónimo o frase memorable
- Una historia vívida vinculando los elementos
- Una rima o canción
- Asociaciones visuales
```

**Palacio de la memoria**:
```
Ayúdame a crear un palacio de la memoria para recordar [lista de elementos].

Guíame a través de:
- Elegir una ubicación familiar
- Colocar cada elemento en un lugar específico
- Crear asociaciones vívidas y memorables
- Caminar por el palacio
```

## Aplicación Práctica

**Aprendizaje basado en proyectos**:
```
Quiero aprender [habilidad] construyendo algo. Soy principiante.

Sugiere 5 proyectos para principiantes que:
- Enseñen conceptos fundamentales
- Sean completables en 1-2 semanas
- Se construyan uno sobre otro
- Resulten en algo útil/compartible
```

**Conexiones del mundo real**:
```
Estoy aprendiendo sobre [concepto abstracto como derivadas/oferta y demanda/ADN].

Muéstrame 5 ejemplos del mundo real de dónde se aplica esto en:
- Vida diaria
- Eventos actuales
- Tecnología
- Negocios
- Naturaleza
```

## Superando Bloqueos de Aprendizaje

**Diferentes ángulos**:
```
He intentado aprender [tema] 3 veces y aún no lo entiendo.

Explícalo usando:
1. Un enfoque visual/diagrama
2. Una actividad práctica
3. Una historia del mundo real
4. Una comparación con algo que ya conozco
5. El ángulo "por qué debería importarme"
```

**Depurando la comprensión**:
```
Creo que entiendo [concepto], pero no estoy seguro.

Hazme preguntas para probar mi comprensión e identificar brechas. Empieza básico y ve progresivamente más difícil. Dime dónde está mal mi modelo mental.
```

## Grupos de Estudio y Discusión

**Preguntas de discusión**:
```
Crea 5 preguntas de discusión provocadoras sobre [tema] para mi grupo de estudio:

- Abiertas, sin una sola respuesta "correcta"
- Fomentar el debate
- Conectar con problemas del mundo real
- Ayudarnos a pensar críticamente
```

## Estrategias de Examen

**Análisis de respuestas**:
```
En mi último examen, me equivoqué en esta pregunta:

Pregunta: [pega]
Mi respuesta: [tu respuesta]
Respuesta correcta: [respuesta correcta]

Ayúdame a entender:
- ¿Por qué estaba equivocado?
- ¿Qué estaba pensando vs. qué debería haber pensado?
- ¿Cómo puedo evitar este error la próxima vez?
```
