# Trucos de Productividad con IA

## Trucos de Flujo de Trabajo Diario

### Rutina Matutina
**Truco 1: Resumen diario**
```
Prompt: "Crea mi resumen diario:
- Las 3 prioridades principales de hoy
- Clima y tiempo de viaje
- Preparación de reuniones clave
- Noticias en [tu industria]"
```

**Truco 2: Clasificación de emails**
```
Prompt: "Resume estos 20 emails y prioriza:
[pega emails]

Para cada uno: urgencia, acción requerida, ¿puede esperar?"
```

### Durante el Día

**Truco 3: Notas de reunión a elementos de acción**
```
Prompt: "Extrae elementos de acción de estas notas de reunión:
[pega notas]

Formato:
- Tarea
- Responsable
- Fecha de entrega
- Prioridad"
```

**Truco 4: Investigación rápida**
```
Prompt: "Resumen de investigación de 5 minutos sobre [tema]:
- Datos clave
- Desarrollos recientes
- Por qué importa
- 3 fuentes imprescindibles"
```

**Truco 5: Redacción de mensajes**
```
Prompt: "Reescribe este mensaje para que sea:
- Más profesional
- Conciso (menos de 100 palabras)
- Orientado a la acción

Original: [pega mensaje]"
```

### Final del Día

**Truco 6: Resumen diario**
```
Prompt: "Revisa mi día y crea:
- Resumen de logros
- Las 3 principales de mañana
- Seguimientos necesarios
- Pregunta de reflexión"
```

## Trucos de Comunicación

### Email
- **Respuestas rápidas**: "Redacta una declinación educada a [solicitud de reunión]"
- **Seguimientos**: "Escribe un seguimiento gentil a mi email de la semana pasada sobre [tema]"
- **Introducciones**: "Redacta un email de introducción conectando a [Persona A] con [Persona B] sobre [tema]"

### Slack/Teams
- **Actualizaciones de estado**: "Crea una actualización de estado resumiendo: [puntos]"
- **Anuncios**: "Haz este anuncio más atractivo: [texto]"

### Conversaciones Difíciles
```
Prompt: "Ayúdame a comunicar este mensaje difícil:
Situación: [describe]
Necesito transmitir: [puntos clave]
Mantener: [profesional/empático/firme]

Proporciona 3 enfoques"
```

## Trucos de Aprendizaje

**Aprendizaje rápido**:
```
Prompt: "Enséñame [tema] usando:
- Explicación ELI5
- Principios clave (3-5)
- Ejemplos prácticos
- Errores comunes
- Siguientes pasos para aprender más"
```

**Resúmenes de libros**:
```
Prompt: "Resume [nombre del libro]:
- Tesis principal
- Argumentos clave
- Conclusiones accionables
- Perspectiva crítica"
```

## Trucos Creativos

**Lluvia de ideas**:
```
Prompt: "10 ideas creativas para [problema/proyecto].
Mezcla: convencional + no convencional
Para cada una: breve descripción + viabilidad"
```

**Bloqueo de escritor**:
```
Prompt: "Estoy atascado escribiendo sobre [tema].
Dame:
- 5 ángulos diferentes
- Ganchos de apertura
- Preguntas interesantes para explorar"
```

## Trucos de Planificación

**Inicio de proyecto**:
```
Prompt: "Ayuda a planificar [proyecto]:
- Dividir en fases
- Hitos clave
- Riesgos potenciales
- Necesidades de recursos
- Estimación de cronograma"
```

**Marcos de decisión**:
```
Prompt: "Ayúdame a decidir: [decisión].
Crea:
- Análisis de pros/contras
- Evaluación de riesgos
- Escenarios (mejor/peor/probable)
- Recomendación"
```

## Trucos de Vida Personal

**Planificación de comidas**:
```
Prompt: "Semana de cenas saludables:
- Presupuesto: $[cantidad]
- Dietético: [preferencias]
- Tiempo de cocina: [máximo minutos]
- Lista de compras incluida"
```

**Planificación de viajes**:
```
Prompt: "Itinerario de 3 días para [ciudad]:
- Intereses: [lista]
- Presupuesto: [nivel]
- Imprescindibles + joyas ocultas
- Plan día a día
- Recomendaciones de restaurantes"
```

**Ideas de regalos**:
```
Prompt: "Ideas de regalos para [persona]:
- Edad: [edad]
- Intereses: [lista]
- Presupuesto: $[cantidad]
- Ocasión: [ocasión]
Incluye: por qué encaja, dónde comprar"
```

## Trucos de Automatización

**Crear GPT personal**:
- Instrucciones personalizadas para tu rol
- Ahorra tiempo en prompts repetidos
- Estilo de salida consistente

**Atajos de extensión de navegador**:
- GPT para Sheets
- ChatGPT Writer (emails)
- Merlin (cualquier página web)

**Voz a texto**:
- Usa voz de ChatGPT en teléfono
- Dicta ideas mientras viajas
- Lluvia de ideas manos libres

## Mejores Prácticas

✅ **Hacer**:
- Guardar tus mejores prompts
- Iterar en salidas
- Combinar múltiples trucos
- Personalizar para tus necesidades
- Compartir victorias con equipo

❌ **No hacer**:
- Copiar-pegar sin leer
- Usar para información sensible
- Olvidar verificar datos
- Depender excesivamente de IA
- Omitir el toque humano

## Biblioteca de Prompts Inicial

Crea tu propia biblioteca de prompts:

1. **Carpeta de plantillas**:
   - Plantillas de email
   - Preparación de reuniones
   - Resúmenes de investigación
   - Marcos de planificación

2. **Contextos personales**:
   - Tu rol/industria
   - Tareas comunes
   - Nombres de equipo/proyectos
   - Información usada frecuentemente

3. **Control de versiones**:
   - Nota qué funciona
   - Refina con el tiempo
   - Rastrea tiempo ahorrado

## Estimación de Ahorro de Tiempo

**Ahorro diario promedio**:
- Email: 30 mins
- Reuniones: 15 mins
- Investigación: 20 mins
- Escritura: 25 mins
- Planificación: 15 mins

**Total**: ~2 horas/día = 10 horas/semana
