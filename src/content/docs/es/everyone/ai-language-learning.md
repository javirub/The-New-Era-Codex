# IA para Aprendizaje de Idiomas

## Práctica de Conversación

```
Prompt: Tengamos una conversación en [español]:

Mi nivel: [Principiante/Intermedio/Avanzado]
Tema: [Vida diaria/Viajes/Negocios]

Reglas:
- Habla a mi nivel
- Corrige mis errores en [corchetes]
- Explica gramática cuando pregunte
- Mezcla formal/informal apropiadamente
```

## Explicación de Gramática

```
Prompt: Explica [concepto gramatical] en [idioma]:

Usa:
- Términos simples
- Ejemplos
- Cuándo usarlo
- Errores comunes
- Oraciones de práctica
```

## Traducción y Contexto

```
Prompt: Traduce y explica:

Frase: [frase en idioma objetivo]

Proporciona:
- Traducción literal
- Traducción natural
- Cuándo usarla
- Contexto cultural
- Expresiones similares
```

## Construcción de Vocabulario

```
Prompt: Enséñame 20 palabras de [tema] en [idioma]:

Tema: [Comida/Viajes/Negocios]
Nivel: [Tu nivel]

Para cada:
- Palabra
- Pronunciación
- Oración de ejemplo
- Consejo de memoria
```

## Práctica de Escritura

```
Prompt: Revisa mi escritura en [idioma]:

Mi texto: [pega]

Verifica:
- Gramática
- Vocabulario
- Fluidez natural
- Apropiación cultural

Proporciona correcciones con explicaciones.
```

## Herramientas

- **Duolingo**: Lecciones con IA
- **ChatGPT**: Compañero de conversación
- **ElevenLabs**: Práctica de pronunciación
- **DeepL**: Traducción avanzada
