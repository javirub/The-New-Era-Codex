# Comenzando con IA

¿Nuevo en IA? ¡Comienza aquí! Estas guías te enseñarán lo básico en lenguaje simple y sin jerga. No se requiere experiencia técnica.

<Aside type="tip" title="Perfecto para Principiantes">
Estas guías no asumen conocimiento previo. Si puedes usar un navegador web, puedes seguir estos tutoriales.
</Aside>

## Tu Ruta de Aprendizaje

<Steps>
1. **Comienza con ChatGPT**
   Empieza con la [Guía de ChatGPT para Principiantes](/es/everyone/getting-started/chatgpt-beginners-guide) para aprender lo básico de IA conversacional.

2. **Aprende Mejores Prompts**
   Usa [Plantillas de Prompts Mágicos](/es/everyone/getting-started/magic-prompt-templates) para obtener mejores resultados de cualquier herramienta de IA.

3. **Explora Más**
   Revisa [Recursos de Aprendizaje de IA](/es/everyone/getting-started/ai-learning-resources) para continuar tu viaje con IA.

4. **Mantente Seguro**
   Aprende sobre [Detección de Contenido de IA](/es/everyone/getting-started/detecting-ai-content) para reconocer texto generado por IA.
</Steps>
