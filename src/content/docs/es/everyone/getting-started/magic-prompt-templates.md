# 50 Plantillas de Prompts Mágicos: Copia, Pega, Obtén Resultados

## Cómo Usar Esta Guía

Cada plantilla tiene:
- **Propósito**: Para qué sirve
- **Plantilla**: Lista para copiar y pegar (reemplaza partes en [corchetes])
- **Ejemplo**: Véla en acción

**Consejo pro**: Guarda tus favoritas y personalízalas para tus necesidades.

## Escritura y Comunicación (10 Plantillas)

### 1. Escritor de Correos Profesionales

**Propósito**: Escribir cualquier correo de negocios rápidamente

**Plantilla**:
```
Escribe un correo profesional a [destinatario] sobre [tema].
Tono: [formal/amigable/disculpa]
Puntos clave a incluir:
- [punto 1]
- [punto 2]
- [punto 3]
Manténlo bajo [conteo palabras] palabras.
```

**Ejemplo**:
```
Escribe un correo profesional a mi equipo sobre la extensión del plazo del proyecto.
Tono: amigable pero claro
Puntos clave a incluir:
- Nueva fecha límite es 15 de marzo
- Razones para extensión (retrasos en feedback del cliente)
- Qué lograremos con tiempo extra
Manténlo bajo 150 palabras.
```

### 2. Mejorador de Texto

**Propósito**: Hacer cualquier texto mejor

**Plantilla**:
```
Mejora este texto haciéndolo más [profesional/casual/persuasivo/claro]:

[pega tu texto]

Mantén el mismo significado pero mejora [tono/claridad/impacto].
```

### 3. Revisor de Gramática y Estilo

**Propósito**: Corregir errores y mejorar estilo

**Plantilla**:
```
Revisa este texto para errores de gramática, ortografía y estilo.
Proporciona la versión corregida y lista los cambios hechos:

[pega tu texto]
```

### 4. Generador de Actas de Reunión

**Propósito**: Crear notas estructuradas desde notas crudas de reunión

**Plantilla**:
```
Convierte estas notas de reunión en actas profesionales:

[pega tus notas]

Incluye:
- Asistentes
- Decisiones clave
- Ítems de acción con responsables
- Próximos pasos
```

### 5. Creador de Posts de LinkedIn

**Propósito**: Posts sociales atractivos

**Plantilla**:
```
Crea un post de LinkedIn sobre [tema].
Ángulo: [historia personal/insight de industria/logro]
Tono: [profesional/inspiracional/conversacional]
Incluye: gancho, mensaje principal, llamado a la acción
Longitud: [150-300 palabras]
```

### 6. Carta de Disculpa

**Propósito**: Disculpas profesionales

**Plantilla**:
```
Escribe una carta de disculpa por [situación].
Para: [destinatario]
Expresa: remordimiento genuino, entendimiento del impacto
Ofrece: [solución específica/compensación]
Tono: sincero y profesional
```

### 7. Nota de Agradecimiento

**Propósito**: Expresar gratitud profesionalmente

**Plantilla**:
```
Escribe una nota de agradecimiento a [persona] por [razón].
Menciona específicamente: [qué apreciaste]
Tono: [cálido/formal/casual]
Longitud: [corto/medio/largo]
```

### 8. Resumen Ejecutivo

**Propósito**: Resumir documentos largos

**Plantilla**:
```
Crea un resumen ejecutivo de este documento:

[pega texto o describe documento]

Formato: 3-5 puntos
Enfoque en: hallazgos clave, recomendaciones, ítems de acción
Audiencia objetivo: [ejecutivos/equipo/clientes]
```

### 9. Mensaje de Alcance Frío

**Propósito**: Primer contacto con prospectos/socios

**Plantilla**:
```
Escribe un mensaje de alcance frío a [persona objetivo/empresa].
Mi oferta: [qué proporcionas]
Su beneficio: [por qué deberían importarles]
Llamado a la acción: [qué quieres que hagan]
Tono: profesional pero personal
Longitud: menos de 100 palabras
```

### 10. Script de Presentación

**Propósito**: Notas para hablar en presentaciones

**Plantilla**:
```
Escribe un script de presentación para una charla de [duración] minutos sobre [tema].
Audiencia: [quiénes son]
Mensajes clave:
- [mensaje 1]
- [mensaje 2]
- [mensaje 3]
Incluye: gancho de apertura, transiciones, llamado a la acción de cierre
```

## Aprendizaje y Educación (10 Plantillas)

### 11. Explica Como Si Tuviera 5 Años

**Propósito**: Entender conceptos complejos simplemente

**Plantilla**:
```
Explica [concepto complejo] como si tuviera [edad/nivel de experiencia].
Usa: palabras simples, analogías, ejemplos de [dominio relevante]
Evita: jerga, términos técnicos
```

**Ejemplo**:
```
Explica computación cuántica como si tuviera 10 años.
Usa: palabras simples, analogías, ejemplos de la vida cotidiana
Evita: jerga, términos técnicos
```

### 12. Creador de Rutas de Aprendizaje

**Propósito**: Planes de aprendizaje estructurados

**Plantilla**:
```
Crea una ruta de aprendizaje de [período] para [habilidad/tema].
Nivel actual: [principiante/intermedio/avanzado]
Tiempo disponible: [horas por semana]
Objetivo: [qué quieres lograr]
Incluye: temas semanales, recursos, ejercicios de práctica
```

### 13. Generador de Guías de Estudio

**Propósito**: Preparación para exámenes

**Plantilla**:
```
Crea una guía de estudio para [materia/tema].
Fecha del examen: [fecha]
Formato: [opción múltiple/ensayo/práctico]
Incluye:
- Resumen de conceptos clave
- 10 preguntas de práctica
- Errores comunes a evitar
```

### 14. Comparación de Conceptos

**Propósito**: Entender diferencias

**Plantilla**:
```
Compara y contrasta [concepto A] vs [concepto B].
Formato: tabla comparativa
Categorías: definición, casos de uso, pros, contras, mejor para
Audiencia: [tu nivel]
```

### 15. Aplicaciones del Mundo Real

**Propósito**: Conectar teoría con práctica

**Plantilla**:
```
Dame 5 aplicaciones del mundo real de [concepto/teoría].
Industria: [especifica o di "cualquiera"]
Incluye: descripción breve, empresas que lo usan, por qué funciona
```

### 16. Tutorial Paso a Paso

**Propósito**: Aprender haciendo

**Plantilla**:
```
Crea un tutorial paso a paso para [tarea/habilidad].
Formato: pasos numerados con explicaciones
Incluye: prerrequisitos, errores comunes, consejos
Asume: [nivel de conocimiento]
```

### 17. Generador de Cuestionarios

**Propósito**: Prueba tu conocimiento

**Plantilla**:
```
Crea un cuestionario de [número] preguntas sobre [tema].
Dificultad: [fácil/media/difícil]
Formato: [opción múltiple/verdadero-falso/respuesta corta]
Incluye clave de respuestas con explicaciones
```

### 18. Creador de Tarjetas de Memoria

**Propósito**: Ayuda para memorización

**Plantilla**:
```
Crea 20 tarjetas de memoria para [tema].
Formato: Pregunta en frente, Respuesta atrás
Enfócate en: [términos clave/conceptos/fórmulas]
```

### 19. Resumen de Investigación

**Propósito**: Insights rápidos de artículos

**Plantilla**:
```
Resume esta investigación/artículo en 5 puntos:

[pega texto o descripción de URL]

Enfócate en: metodología, hallazgos clave, implicaciones
Objetivo: alguien con [nivel de experiencia]
```

### 20. Organizador de Notas

**Propósito**: Estructurar notas desordenadas

**Plantilla**:
```
Organiza estas notas de estudio en una estructura clara:

[pega tus notas]

Crea: temas principales, subtemas, puntos clave
Formato: esquema jerárquico
```

## Resolución de Problemas y Planificación (10 Plantillas)

### 21. Matriz de Decisión

**Propósito**: Tomar mejores decisiones

**Plantilla**:
```
Ayúdame a decidir entre [opción A] y [opción B] para [situación].
Crea una matriz de decisión comparando:
- Costo
- Tiempo requerido
- Beneficios
- Riesgos
- Impacto a largo plazo
Recomienda mejor opción con razonamiento.
```

### 22. Análisis de Pros y Contras

**Propósito**: Evaluar opciones

**Plantilla**:
```
Lista pros y contras de [decisión/opción].
Contexto: [tu situación]
Considera: [factores específicos importantes para ti]
Conclusión: recomendación basada en análisis
```

### 23. Planificador de Proyectos

**Propósito**: Organizar proyectos complejos

**Plantilla**:
```
Crea un plan de proyecto para [nombre del proyecto].
Objetivo: [qué quieres lograr]
Fecha límite: [fecha]
Recursos: [qué tienes]
Incluye: fases, hitos, cronograma, entregables
```

### 24. Guía de Solución de Problemas

**Propósito**: Resolver problemas técnicos

**Plantilla**:
```
Tengo este problema: [describe problema]
Contexto: [cuándo pasa, qué has probado]
Crea una lista de verificación de solución de problemas desde causas más a menos probables.
Incluye: pasos específicos para probar cada solución
```

### 25. Marco de Establecimiento de Objetivos

**Propósito**: Establecer objetivos alcanzables

**Plantilla**:
```
Ayúdame a establecer un objetivo SMART para [qué quieres lograr].
Cronograma: [período]
Situación actual: [dónde estás]
Resultado deseado: [dónde quieres estar]
Incluye: hitos específicos y métricas de éxito
```

### 26. Evaluación de Riesgos

**Propósito**: Identificar y mitigar riesgos

**Plantilla**:
```
Analiza riesgos para [proyecto/decisión].
Identifica:
- Riesgos potenciales
- Probabilidad (alta/media/baja)
- Impacto (alto/medio/bajo)
- Estrategias de mitigación
Formato: matriz de riesgo
```

### 27. Optimizador de Procesos

**Propósito**: Mejorar flujos de trabajo

**Plantilla**:
```
Analiza este flujo de trabajo y sugiere mejoras:

Proceso actual:
[describe pasos actuales]

Objetivos: [más rápido/más barato/mejor calidad]
Restricciones: [limitaciones]

Proporciona: proceso optimizado, ahorros de tiempo/costo, pasos de implementación
```

### 28. Compañero de Lluvia de Ideas

**Propósito**: Generar ideas

**Plantilla**:
```
Lluvia de ideas de [número] ideas creativas para [problema/oportunidad].
Contexto: [antecedentes relevantes]
Restricciones: [presupuesto/tiempo/recursos]
Para cada idea proporciona: descripción, pros, puntuación de viabilidad
```

### 29. Matriz de Prioridades

**Propósito**: Enfocarse en lo que importa

**Plantilla**:
```
Ayúdame a priorizar estas tareas:
[lista tus tareas]

Usa: Matriz de Eisenhower (urgente/importante)
Proporciona: lista categorizada, orden sugerido, razonamiento
```

### 30. Análisis de Causa Raíz

**Propósito**: Encontrar problemas reales

**Plantilla**:
```
Realiza un análisis de causa raíz para este problema:
[describe el problema]

Usa: técnica de los 5 Porqués
Identifica: causa raíz, factores contribuyentes, soluciones
```

## Consejos Pro para Todas las Plantillas

### 1. Estratifica Tus Prompts

Empieza amplio, luego refina:
```
Primero: [Usa plantilla]
Luego: "Hazlo más formal"
Luego: "Añade una sección sobre X"
Luego: "Acorta a 200 palabras"
```

### 2. Añade Ejemplos

```
[Plantilla] +

Como este ejemplo: [pega ejemplo]
Empareja el estilo pero cambia el contenido.
```

### 3. Especifica Formato

```
[Plantilla] +

Formato de salida como:
- Puntos
- Lista numerada
- Tabla
- Markdown
```

### 4. Solicita Variaciones

```
[Plantilla] +

Dame 3 versiones diferentes:
1. Formal
2. Casual
3. Creativo
```

### 5. Itera

No aceptes la primera salida si no es perfecta:
```
"Bien, pero ¿puedes hacerlo [solicitud más específica]"
```
