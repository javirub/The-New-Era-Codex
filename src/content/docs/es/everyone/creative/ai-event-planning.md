# IA para Planificación de Eventos

## Conceptualización del Evento

```
Prompt: Planifica [tipo de evento]:

Evento: [Cumpleaños/Boda/Conferencia]
Invitados: [Número]
Presupuesto: $[cantidad]
Fecha: [Fecha]
Lugar: [Tipo/Ubicación]
Tema: [Si aplica]

Proporciona:
- Ideas de tema
- Sugerencias de lugar
- Desglose de presupuesto
- Cronograma
- Tipos de proveedores necesarios
```

## Gestión de Lista de Invitados

```
Prompt: Ayuda a organizar lista de invitados:

Evento: [Tipo]
Capacidad total: [Número]
Categorías: [Familia/Amigos/Trabajo/etc]

Crea:
- Prioridad de invitación
- Ideas de distribución de asientos
- Sistema de seguimiento dietético
- Plan de gestión de confirmaciones
```

## Planificación de Menú

```
Prompt: Crea menú:

Evento: [Tipo]
Invitados: [Número]
Necesidades dietéticas: [Lista]
Estilo: [Buffet/Emplazado/Cóctel]
Presupuesto: $[por persona]
Temporada: [Temporada]

Sugiere:
- Aperitivos
- Platos principales
- Postres
- Bebidas
- Cantidades
```

## Cronograma y Lista de Verificación

```
Prompt: Cronograma de planificación de evento:

Evento: [Tipo]
Fecha: [Fecha]
Complejidad: [Simple/Moderada/Compleja]

Crea lista de verificación:
- 6 meses antes
- 3 meses antes
- 1 mes antes
- 1 semana antes
- Día anterior
- Día del evento
```

## Investigación de Proveedores

```
Prompt: Encuentra proveedores para:

Evento: [Tipo]
Ubicación: [Ciudad]
Presupuesto: [Rango]
Estilo: [Preferencias]

Lista tipos de proveedores necesarios:
- Qué proporcionan
- Preguntas que hacer
- Costos típicos
- Cronograma de reserva
```

## Ideas de Decoración

```
Prompt: Plan de decoración:

Evento: [Tipo]
Tema: [Tema]
Lugar: [Descripción]
Presupuesto: $[cantidad]
Dispuesto a hacer manualidades: [Sí/No]

Sugiere:
- Esquema de colores
- Centros de mesa
- Iluminación
- Decoración de entrada
- Áreas de fotos
- Recomendaciones de hacer vs comprar
```

## Redacción de Invitaciones

```
Prompt: Escribe invitación:

Evento: [Tipo]
Formalidad: [Casual/Formal]
Detalles clave:
- Fecha/hora
- Ubicación
- Código de vestimenta
- Información de confirmación

Incluye redacción apropiada y etiqueta.
```

## Horario del Día del Evento

```
Prompt: Crea cronograma del día:

Evento: [Boda/Fiesta]
Inicio: [Hora]
Duración: [Horas]
Momentos clave: [Lista]

Crea:
- Horario minuto a minuto
- Coordinación de proveedores
- Tiempo de preparación
- Tiempo de contingencia
- Planes de contingencia
```

## Seguimiento de Presupuesto

```
Prompt: Rastrea presupuesto del evento:

Presupuesto total: $[cantidad]
Categorías: [Lugar/Comida/etc]

Crea:
- Asignaciones por categoría
- Hoja de cálculo de seguimiento
- Ideas para ahorrar costos
- Clasificación de prioridades
- Fondo de contingencia
```

## Notas de Agradecimiento

```
Prompt: Escribe notas de agradecimiento:

Evento: [Tipo]
Destinatario: [Relación]
Regalo: [Lo que dieron/hicieron]
Tono: [Cálido/Formal]

Personaliza mientras mantienes la gratitud.
```
