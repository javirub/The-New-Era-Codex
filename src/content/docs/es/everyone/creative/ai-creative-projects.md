# IA para Proyectos Creativos

## Escritura de Historias

**Inicio de historia**:
```
Escribe el párrafo de apertura para una novela de misterio ambientada en el París de 1920.

Escenario: Un club de jazz
Personaje principal: Un detective privado
Tono: Noir, atmosférico
Gancho: Una pintura desaparecida
```

**Desarrollo de personajes**:
```
Crea un perfil detallado de personaje:

Nombre: [tú eliges]
Rol: Villano en una novela de fantasía
Motivación: Quiere restaurar la magia al mundo
Pistas de trasfondo: Pasado trágico, perdió a alguien
Personalidad: Complejo, no puramente malvado

Incluye: Antecedentes, rasgos de personalidad, peculiaridades, miedos, objetivos
```

**Giros de trama**:
```
Estoy escribiendo un thriller donde un detective investiga un asesinato. Sospechosos actuales:
1. El socio comercial de la víctima
2. El ex cónyuge de la víctima
3. Un vecino misterioso

Sugiere 3 giros de trama inesperados que sorprenderían a los lectores.
```

## Poesía y Letras de Canciones

**Creación de poesía**:
```
Escribe un haiku sobre el otoño en la ciudad. Enfócate en: hojas cayendo, calles ocupadas, clima refrescante.
```

**Letras de canciones**:
```
Escribe un estribillo para una canción pop alegre sobre:
- Tema: Nuevos comienzos
- Estado de ánimo: Esperanzador, enérgico
- Estilo: Similar a Taylor Swift
- 4-6 líneas
```

**Asistencia con rimas**:
```
Estoy escribiendo letras y necesito palabras que rimen con "sueño" y se ajusten al tema de "ambición y esperanza". Dame 10 opciones con significados.
```

## Prompts de Arte Visual

**Prompts de generación de imágenes** (para Midjourney, DALL-E, Stable Diffusion):

```
Crea un prompt detallado para generar:
- Un paisaje surrealista
- Combinando: Ciudad submarina se encuentra con ruinas antiguas
- Estilo de arte: Pintura digital, colores vibrantes
- Estado de ánimo: Misterioso, onírico
- Iluminación: Bioluminiscente

Hazlo lo suficientemente detallado para generación de imágenes con IA.
```

**Resultado**:
> "Una antigua ciudad submarina bioluminiscente con templos de piedra cubiertos de coral, cardúmenes de peces brillantes nadando entre columnas de mármol en ruinas, rayos de luz solar filtrada atravesando la superficie del agua, tonos turquesa y púrpura vibrantes, atmósfera misteriosa, pintura digital, muy detallada, tendencia en artstation, 4k"

## Ideas de Diseño Gráfico

**Conceptos de logotipo**:
```
Lluvia de ideas de 5 conceptos de logotipo para:
- Negocio: Productos de limpieza ecológicos
- Nombre: "Green Gleam"
- Valores: Sostenibilidad, efectividad, simplicidad
- Audiencia objetivo: Propietarios de viviendas conscientes del medio ambiente

Para cada concepto, describe: Elementos visuales, colores, estilo
```

**Paleta de colores**:
```
Sugiere una paleta de colores para un sitio web sobre:
- Tema: Meditación y mindfulness
- Sensación: Calma, pacífica, arraigada
- Necesidad: 5 colores (primario, secundario, acentos)

Incluye códigos hexadecimales y describe cuándo usar cada color.
```

## Composición Musical

**Progresiones de acordes**:
```
Sugiere 3 progresiones de acordes para una balada triste y emocional.

Tonalidad: Do mayor o La menor
Estilo: Basado en piano
Vibra: Melancólico, final esperanzador
```

**Ideas de melodía**:
```
Tengo estos acordes: Am - F - C - G

Sugiere un patrón de melodía vocal que funcione bien. Descríbelo en términos de:
- Movimiento (arriba/abajo/sostenido)
- Patrón rítmico
- Dónde colocar énfasis
```

## Ideas de Contenido de Video

**Scripts de videos de YouTube**:
```
Crea un bosquejo de script para un video de YouTube de 10 minutos:

Tema: "5 Comidas Fáciles para Noches Ocupadas de Semana"
Audiencia objetivo: Padres que trabajan
Tono: Amigable, útil, enérgico
Estructura: Intro, 5 recetas (2 min cada una), conclusión

Incluye: Gancho, transiciones, llamado a la acción
```

**Textos para redes sociales**:
```
Escribe 3 textos de Instagram para una foto de pan casero de masa madre:

Opciones de estilo:
1. Humorístico y relatable
2. Educativo (consejos de horneado)
3. Inspiracional/vibra acogedora

Cada uno 150-200 caracteres, incluye hashtags relevantes
```

## Planificación de Podcasts

**Bosquejos de episodios**:
```
Crea un bosquejo de episodio de podcast:

Tema: "La Psicología de la Procrastinación"
Duración: 30 minutos
Formato: Comentario solo + consejos prácticos
Audiencia: Jóvenes profesionales

Incluye: Intro, 3 segmentos principales, ejemplos, conclusiones accionables
```

**Preguntas de entrevista**:
```
Genera 10 preguntas reflexivas de entrevista para un invitado que es:
- Un autor bestseller
- Escribe novelas de thriller
- Superó la dislexia para convertirse en escritor

Mezcla de: viaje personal, consejos de oficio, perspectivas de la industria
```

## Proyectos de Fotografía

**Ideas de proyectos fotográficos**:
```
Sugiere 5 temas de proyectos fotográficos que:
- Puedan hacerse en un entorno urbano
- No requieran equipo especial
- Tengan profundidad artística
- Sean buenos para un principiante mejorando sus habilidades

Para cada uno, explica el concepto y 3 tomas específicas a capturar.
```

**Guías de estilo de edición**:
```
Describe el estilo de edición para lograr un aspecto "sombrío, cinemático" para fotografía urbana:
- Temperatura de color
- Niveles de contraste
- Ajustes de sombra/luz
- Sugerencias de gradación de color
```

## Manualidades y Bricolaje

**Instrucciones de proyecto**:
```
Dame instrucciones paso a paso para hacer:
- Un colgante de pared de macramé
- Para principiantes
- Tiempo de proyecto de 2-3 horas
- Lista de materiales incluida
```

**Ideas de reciclaje**:
```
Tengo 10 botellas de vino vacías. Sugiere 5 proyectos de bricolaje creativos para reutilizarlas en:
- Decoración del hogar
- Artículos funcionales
- Regalos

Para cada idea, incluye nivel de dificultad y materiales necesarios.
```

## Calendario de Contenido

**Planificación de redes sociales**:
```
Crea un calendario de contenido de Instagram de 2 semanas para:
- Cuenta: Panadería pequeña
- Objetivo: Mostrar productos, construir comunidad
- Frecuencia de publicación: 1 publicación diaria
- Mezcla de contenido: 60% productos, 30% detrás de escenas, 10% destacados de clientes

Proporciona: Tema para cada día, ideas de texto, estrategia de hashtags
```

## Lluvia de Ideas y Generación de Ideas

**Rompedores de bloqueo creativo**:
```
Estoy atascado en un proyecto creativo (pintando un paisaje). Dame 10 enfoques inusuales o restricciones para generar nuevas ideas:

Ejemplos: Pintar con solo 3 colores, pintar al revés, usar herramientas inusuales, etc.
```

**Combinaciones de conceptos**:
```
Combina estos dos conceptos improbables en un proyecto creativo único:
1. [Caballeros medievales]
2. [Exploración espacial]

Sugiere: Idea de historia, proyecto de arte, concepto de video o tema musical basado en esta combinación
```

## Construcción de Mundos (Ficción)

**Creación de mundo de fantasía**:
```
Ayúdame a crear un mundo de fantasía:

Concepto central: Magia impulsada por emociones
Conflicto: La magia se está desvaneciendo a medida que la sociedad se vuelve más lógica
Geografía: Archipiélago de islas

Desarrolla: Sistemas de gobierno, reglas de magia, conflictos culturales, ubicaciones clave
```

**Generadores de nombres**:
```
Genera 10 nombres de personajes de fantasía únicos que suenen:
- Élficos/etéreos
- Fuertes y memorables
- No demasiado difíciles de pronunciar

Para cada uno, sugiere qué tipo de personaje encaja (guerrero, mago, etc.)
```
