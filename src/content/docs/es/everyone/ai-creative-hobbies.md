# IA para Pasatiempos Creativos

## Escritura Creativa

### Ideas para Historias
```
Prompt: Genera ideas para historias:

Género: [Misterio/Romance/Ciencia Ficción]
Temas: [Lista]
Longitud: [Cuento corto/Novela]
Giro único: [Restricción opcional]

Dame 10 conceptos.
```

### Desarrollo de Personajes
```
Prompt: Desarrolla un personaje:

Rol: [Protagonista/Antagonista]
Género: [Género]
Arquetipo: [Punto de partida]

Crea:
- Trasfondo
- Rasgos de personalidad
- Motivaciones
- Defectos
- Arco del personaje
- Estilo de diálogo
```

### Retroalimentación de Escritura
```
Prompt: Critica este pasaje:

[Pega tu escritura]

Retroalimentación sobre:
- Ritmo
- Naturalidad del diálogo
- Mostrar vs contar
- Descripciones
- Impacto emocional

Sé constructivo.
```

## Arte Visual

### Ideas de Composición
```
Prompt: Sugiere composiciones para:

Sujeto: [Paisaje/Retrato/Naturaleza muerta]
Estilo: [Realista/Abstracto/Impresionista]
Ambiente: [Calmo/Dramático/Caprichoso]

Describe 5 composiciones interesantes.
```

### Paletas de Colores
```
Prompt: Crea paleta de colores:

Tema: [Atardecer/Océano/Otoño]
Colores: [Número]
Estilo: [Cálidos/Fríos/Complementarios]

Proporciona códigos hexadecimales y sugerencias de uso.
```

## Música

### Estructura de Canciones
```
Prompt: Ayúdame a estructurar una canción:

Género: [Pop/Rock/Jazz]
Tema: [Amor/Pérdida/Aventura]
Duración: [Minutos]

Sugiere:
- Estructura de verso/estribillo
- Tempo
- Tonalidad
- Instrumentación
- Temas de letra por sección
```

### Escritura de Letras
```
Prompt: Escribe letras de canción:

Tema: [Tópico]
Estilo: [Artista/Género]
Ambiente: [Feliz/Melancólico]
Duración: [3-4 minutos]

Incluye:
- Versos
- Estribillo
- Puente
- Esquema de rima
```

## Fotografía

### Planificación de Tomas
```
Prompt: Planifica sesión fotográfica:

Sujeto: [Retrato/Paisaje/Producto]
Ubicación: [Descripción]
Hora: [Hora dorada/Mediodía/Hora azul]
Equipo: [Lo que tienes]

Sugiere:
- Lista de tomas
- Ángulos
- Configuración de iluminación
- Ajustes de cámara
- Consejos de composición
```

### Ideas de Edición Fotográfica
```
Prompt: Sugerencias de edición:

Descripción de foto: [Qué está en el encuadre]
Sensación actual: [Ambiente actual]
Sensación deseada: [Ambiente objetivo]
Estilo: [Oscuro/Brillante/Vintage]

Sugiere ajustes para:
- Exposición
- Colores
- Contraste
- Ediciones específicas
```

## Manualidades y Creación

### Ideas de Proyectos
```
Prompt: Ideas de proyectos de manualidades:

Habilidad: [Tejido/Carpintería/etc]
Nivel: [Principiante/Avanzado]
Materiales: [Lo que tienes]
Tiempo: [Horas disponibles]

Sugiere 5 proyectos con calificaciones de dificultad.
```

## Desarrollo de Habilidades

```
Prompt: Ruta de aprendizaje para:

Habilidad: [Dibujo/Guitarra/Fotografía]
Nivel actual: [Principiante]
Tiempo: [Horas por semana]
Objetivos: [Lo que quieres lograr]

Crea:
- Plan de aprendizaje de 3 meses
- Recursos
- Ejercicios de práctica
- Hitos
```
