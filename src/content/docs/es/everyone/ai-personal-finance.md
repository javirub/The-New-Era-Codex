# IA para Finanzas Personales

## Creación de Presupuesto

```
Prompt: Crea presupuesto mensual:

Ingreso: $[cantidad]
Gastos fijos:
- Renta: $[cantidad]
- Servicios: $[cantidad]
- Seguro: $[cantidad]

Objetivos financieros:
- Fondo de emergencia: $[objetivo]
- Retiro: [%]
- Ahorros: [propósito]

Recomienda asignación usando regla 50/30/20.
```

## Análisis de Gastos

```
Prompt: Analiza mis gastos:

Gastos del mes pasado:
[Pega lista o resumen]

Identifica:
- Gastos innecesarios
- Oportunidades de ahorro
- Patrones de gasto
- Optimización de presupuesto
- Elementos de acción
```

## Objetivos de Ahorro

```
Prompt: Crea plan de ahorro:

Objetivo: $[cantidad] para [propósito]
Cronograma: [meses/años]
Ahorros actuales: $[cantidad]
Presupuesto mensual: $[cantidad]

Proporciona:
- Ahorro mensual necesario
- Estrategia de inversión
- Hitos
- Consejos de motivación
```

## Estrategia de Pago de Deudas

```
Prompt: Planifica pago de deudas:

Deudas:
1. Tarjeta de crédito: $[cantidad] al [%] APR
2. Préstamo estudiantil: $[cantidad] al [%] APR
3. Préstamo de auto: $[cantidad] al [%] APR

Disponible mensualmente: $[cantidad]

Compara:
- Método avalancha (tasa más alta primero)
- Método bola de nieve (más pequeña primero)

Recomienda mejor enfoque.
```

## Investigación de Inversiones

```
Prompt: Explica opción de inversión:

Inversión: [ETF/Acción/Fondo nombre]
Mi perfil:
- Edad: [edad]
- Tolerancia al riesgo: [baja/media/alta]
- Objetivos: [retiro/crecimiento/ingreso]
- Cronograma: [años]

Explica:
- Qué es
- Pros/contras
- Ajuste para mi perfil
- Riesgos
- Alternativas

⚠️ No es consejo financiero - ¡investiga!
```

## Educación Financiera

```
Prompt: Explica como si tuviera 5 años:

Concepto: [401k/IRA/interés compuesto/etc]

Incluye:
- Explicación simple
- Por qué importa
- Cómo empezar
- Errores comunes
- Recursos para aprender más
```

## Planificación Fiscal

```
Prompt: Ideas de optimización fiscal:

Ingreso: $[cantidad]
Declaración: [Soltero/Casado]
Estado: [Estado]
Deducciones: [lista actual]

Sugiere:
- Deducciones comúnmente perdidas
- Cuentas con ventajas fiscales
- Estrategias de tiempo
- Necesidades de documentación

⚠️ ¡Consulta a un profesional de impuestos!
```

## Hoja de Trabajo de Objetivos Financieros

```
Prompt: Ayúdame a establecer objetivos financieros:

Situación actual:
- Ingreso: $[cantidad]
- Ahorros: $[cantidad]
- Deuda: $[cantidad]
- Edad: [edad]

Objetivos de vida:
- [Objetivo 1]
- [Objetivo 2]
- [Objetivo 3]

Crea objetivos financieros SMART con cronograma.
```

## Negociación de Facturas

```
Prompt: Ayúdame a negociar esta factura:

Servicio: [Internet/teléfono/seguro]
Costo actual: $[cantidad]
He sido cliente: [X] años
Precios de competidores: $[cantidad]

Escribe:
- Guión para llamada
- Plantilla de email
- Puntos de conversación
- Alternativas si no ceden
```

## Planificación de Fondo de Emergencia

```
Prompt: Plan de fondo de emergencia:

Gastos mensuales: $[cantidad]
Ahorros actuales: $[cantidad]
Objetivo: [3/6/12 meses]
Mensual para ahorrar: $[cantidad]

Proporciona:
- Cronograma al objetivo
- Dónde mantenerlo
- Cuándo usarlo
- Cómo reconstruirlo
```

## Planificación de Retiro

```
Prompt: Planificación de retiro:

Edad actual: [edad]
Edad de retiro: [edad]
Ahorros actuales: $[cantidad]
Contribución mensual: $[cantidad]
Retorno esperado: [%]

Estima:
- Fondo de retiro
- Ingreso mensual de retiro
- Brechas a abordar
- Estrategias de optimización
```

## Verificación de Salud Financiera

```
Prompt: Evalúa mi salud financiera:

Ingreso: $[cantidad]/año
Ahorros: $[cantidad]
Inversiones: $[cantidad]
Deuda: $[cantidad]
Gastos mensuales: $[cantidad]
Fondo de emergencia: [X] meses
Retiro: [Y]% de ingreso

Califica A-F y proporciona:
- Fortalezas
- Debilidades
- Acciones prioritarias
```

## Descargos de Responsabilidad Importantes

⚠️ **La IA NO es**:
- Un asesor financiero licenciado
- Capaz de ver tu situación completa
- Responsable de resultados
- Un reemplazo para profesionales

✅ **La IA PUEDE**:
- Ayudar a organizar pensamientos
- Explicar conceptos
- Generar ideas
- Proporcionar educación general
- Crear plantillas

**Siempre**:
- Verificar información
- Consultar profesionales para decisiones importantes
- Considerar tu situación única
- Usar múltiples fuentes
- Actualizar información regularmente
