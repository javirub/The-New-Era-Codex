# IA para el Desarrollo Profesional

## Currículum y Carta de Presentación

**Optimización del currículum**:
```
Revisa mi currículum para un puesto de [título del trabajo]:

[pega el currículum]

Descripción del trabajo objetivo:
[pega la descripción del trabajo]

Proporciona:
- Coincidencias de palabras clave ATS
- Logros a destacar
- Debilidades a abordar
- Mejoras de formato
- Reescrituras de viñetas (verbos de acción, resultados cuantificables)
```

**Formulación de logros**:
```
Ayúdame a reescribir estas experiencias laborales usando el método "STAR" (Situación, Tarea, Acción, Resultado):

Actual: "Gestioné cuentas de redes sociales"

Hazlo: Cuantificable, impresionante, específico, enfocado en logros
```

**Carta de presentación**:
```
Escribe una carta de presentación para:
- Puesto: [título del trabajo] en [empresa]
- Por qué estoy interesado: [tus razones]
- Calificaciones clave: [tu experiencia]
- Lo que aporto: [valor único]

Tono: Profesional pero personal
Longitud: 3 párrafos
```

## Preparación para Entrevistas

**Preguntas comunes**:
```
Tengo una entrevista para [puesto]. Prepárame para preguntas comunes:

1. "Háblame de ti"
2. "¿Por qué quieres trabajar aquí?"
3. "¿Cuál es tu mayor debilidad?"
4. "¿Dónde te ves en 5 años?"

Para cada una, dame:
- Marco para responder
- Respuesta de muestra adaptada a mi experiencia
- Qué evitar
```

**Práctica del método STAR**:
```
Ayúdame a preparar historias STAR para:
- Una ocasión en que mostré liderazgo
- Un conflicto que resolví
- Un fracaso y qué aprendí
- Una ocasión en que di más de lo esperado

Para cada una, hazme preguntas para extraer detalles, luego ayúdame a estructurar la historia de manera convincente.
```

**Preguntas conductuales**:
```
Genera 20 preguntas de entrevista conductual para un rol de [tipo de trabajo], organizadas por tema:
- Liderazgo
- Resolución de problemas
- Trabajo en equipo
- Adaptabilidad
- Resultados/logros
```

**Preguntas para el entrevistador**:
```
Estoy entrevistándome para [puesto] en [empresa]. Sugiere 10 preguntas reflexivas que debería hacer:

- Sobre el rol
- Sobre la cultura del equipo
- Sobre oportunidades de crecimiento
- Sobre la dirección de la empresa

Que muestren que soy: Comprometido, estratégico, consciente de la cultura
```

## Planificación de Carrera

**Análisis de brechas de habilidades**:
```
Actualmente soy [rol actual] y quiero convertirme en [rol objetivo] en 2 años.

Compara:
- Habilidades que tengo
- Habilidades que necesito
- Cómo adquirir las habilidades faltantes
- Cronograma para cada una
- Certificaciones que vale la pena obtener
```

**Hoja de ruta profesional**:
```
Crea un plan de carrera de 5 años:

Punto de partida: [rol/nivel actual]
Meta: [rol/nivel soñado]
Industria: [tu industria]
Restricciones: [ej., no puedo reubicarme, necesito mantener el salario actual]

Incluye: Hitos, desarrollo de habilidades, objetivos de networking, posibles pivotes
```

## Marca Profesional

**Perfil de LinkedIn**:
```
Optimiza mi titular y resumen de LinkedIn:

Actual: [pega]
Rol: [título del trabajo]
Industria: [industria]
Lo que me hace único: [tus diferenciadores]

Hazlo: Buscable, convincente, auténtico
```

**Declaración de marca personal**:
```
Ayúdame a crear una declaración de marca personal:
- Lo que hago: [rol/experiencia]
- A quién sirvo: [audiencia objetivo]
- Cómo agrego valor: [tu enfoque único]
- Objetivos profesionales: [hacia dónde te diriges]

Mantenlo: Una oración, memorable, auténtico
```

## Networking

**Mensajes de contacto**:
```
Escribe un mensaje de LinkedIn para:
- Quién: Alguien en la empresa de mis sueños
- Propósito: Entrevista informativa
- Conexión: [cómo los encontraste]
- Lo que quieres: Llamada de 20 minutos para aprender sobre su rol

Tono: Profesional, respetuoso de su tiempo, solicitud específica
```

**Discurso de presentación**:
```
Crea un discurso de presentación de 30 segundos:
- Quién soy: [antecedentes]
- Lo que hago: [rol actual]
- Lo que busco: [oportunidades/conexiones]
- Gancho memorable: [hecho o logro interesante]

3 versiones: Networking casual, evento formal, feria de empleo
```

**Seguimiento después de networking**:
```
Conocí a [persona] en [evento] y discutimos [tema]. Escribe un correo de seguimiento que:
- Les agradezca
- Haga referencia a nuestra conversación
- Proporcione valor (artículo/conexión/idea)
- Sugiera el siguiente paso
- No sea insistente
```

## Negociación Salarial

**Investigación y preparación**:
```
Estoy negociando para un rol de [título del trabajo] en [ciudad/industria].

Ayúdame a:
- Investigar el rango salarial justo
- Calcular mi objetivo (basado en experiencia, habilidades, mercado)
- Preparar puntos de negociación
- Planificar diferentes escenarios (oferta baja, no negociable, beneficios en lugar de)
```

**Scripts de negociación**:
```
Crea scripts de respuesta para:

Escenario 1: La oferta es menor de lo esperado
Escenario 2: Preguntan mis expectativas salariales
Escenario 3: Negociar beneficios si el salario es fijo
Escenario 4: Aceptar la oferta profesionalmente

Para cada uno: Qué decir, qué no decir, cómo mantener la relación
```

## Evaluaciones de Desempeño

**Autoevaluación**:
```
Se acerca mi revisión anual. Ayúdame a prepararme:

Logros este año:
- [lista]

Desafíos que superé:
- [lista]

Crea:
- Resumen convincente de mi año
- Declaraciones de impacto cuantificables
- Áreas de crecimiento (formuladas positivamente)
- Objetivos para el próximo año
```

**Solicitar un aumento**:
```
Quiero pedir un aumento. Ayúdame a construir mi caso:

- Tiempo desde el último aumento: [tiempo]
- Nuevas responsabilidades: [lista]
- Logros: [lista]
- Comparación de tarifa de mercado: [investigación]

Crea:
- Declaración de apertura
- Evidencia a presentar
- Monto a solicitar
- Cómo manejar objeciones
```

## Desarrollo de Habilidades

**Plan de aprendizaje**:
```
Necesito aprender [habilidad] para el avance profesional.

Crea un plan de 90 días:
- Qué aprender en qué orden
- Mejores recursos (gratuitos y de pago)
- Proyectos de práctica
- Cómo demostrar esta habilidad
- Cómo agregar al currículum/portafolio
```

**Hoja de ruta de certificaciones**:
```
Estoy en [campo]. ¿Qué certificaciones beneficiarían más mi carrera?

Considera:
- ROI (tiempo y dinero vs. valor)
- Reconocimiento de la industria
- Mis objetivos profesionales
- Demanda actual del mercado

Prioriza y crea cronograma.
```

## Escritura Profesional

**Etiqueta de correo electrónico**:
```
Ayúdame a escribir un correo profesional para:
- Destinatario: [Ejecutivo senior/cliente/colega]
- Propósito: [solicitud/actualización/disculpa]
- Tono: [formal/amigable-profesional]
- Longitud: Mantenerlo conciso

Plantilla: Saludo, contexto, solicitud/mensaje, cierre
```

**Propuestas de negocio**:
```
Bosqueja una propuesta de negocio para:
- Qué: [tu idea/proyecto]
- Problema que resuelve: [cuestión]
- Solución: [enfoque]
- Beneficios: [resultados]
- Recursos necesarios: [presupuesto/tiempo/personas]

Estructura: Resumen ejecutivo, problema, solución, implementación, presupuesto
```

## Desarrollo de Liderazgo

**Escenarios de gestión**:
```
Soy un nuevo gerente enfrentando esta situación:

[describe el desafío con miembro del equipo/proyecto/conflicto]

¿Qué debería:
1. Hacer inmediatamente
2. Evitar hacer
3. Decir (frases específicas)
4. Aprender de esto para el futuro
```

**Marco de delegación**:
```
Tengo dificultades para delegar. Crea un marco para decidir:
- Qué tareas delegar
- A quién
- Cuánta orientación dar
- Cómo hacer seguimiento sin microgestionar
- Cuándo intervenir vs. dejarlos aprender
```

## Transiciones de Carrera

**Plan de cambio de carrera**:
```
Quiero hacer la transición de [industria/rol actual] a [industria/rol objetivo].

Ayúdame a:
- Identificar habilidades transferibles
- Abordar brechas
- Reformular mi experiencia
- Dirigirme a empresas abiertas a cambios de carrera
- Hacer networking efectivamente en la nueva industria
```

**Investigación de industria**:
```
Estoy considerando mudarme a [industria]. Dime:
- Tendencias actuales
- Actores/empresas clave
- Áreas de crecimiento
- Desafíos que enfrenta la industria
- Habilidades demandadas
- Trayectorias profesionales típicas
```

## Equilibrio Vida-Trabajo

**Gestión del tiempo**:
```
Estoy abrumado con el trabajo. Ayúdame a:
- Priorizar mis tareas: [lista todo]
- Identificar qué delegar/eliminar
- Crear un cronograma realista
- Establecer límites
- Decir que no profesionalmente
```

**Prevención del agotamiento**:
```
Me siento agotado. Crea un plan para:
- Evaluar qué lo está causando
- Establecer límites más saludables
- Comunicar necesidades al gerente
- Construir hábitos de trabajo sostenibles
- Saber cuándo es momento de cambiar de trabajo
```

## Trabajos Paralelos y Freelancing

**Configuración de freelance**:
```
Quiero empezar a hacer freelancing como [habilidad/servicio].

Ayúdame a:
- Fijar precio a mis servicios
- Crear un portafolio simple
- Escribir una descripción de servicio
- Encontrar primeros clientes
- Gestionar comunicación con clientes
```

**Ideas de trabajos paralelos**:
```
Dadas mis habilidades: [lista]
Y restricciones: [tiempo disponible, recursos]

Sugiere 5 ideas de trabajos paralelos que:
- Puedan empezar con inversión mínima
- Tengan demanda real del mercado
- Se ajusten a mi horario
- Puedan escalar si tienen éxito
```
