# IA para Proyectos del Hogar

## Planificación de Proyectos

```
Prompt: Planifica renovación del hogar:

Proyecto: [Remodelación de cocina/Baño/Terraza]
Presupuesto: $[cantidad]
Cronograma: [Meses]
Nivel de bricolaje: [Principiante/Intermedio/Experto]
Espacio: [Dimensiones]

Proporciona:
- Desglose por fases
- Lista de materiales
- Estimación de costos
- Cronograma
- Cuándo contratar profesionales
```

## Guías de Bricolaje

```
Prompt: Guía paso a paso:

Tarea: [Instalar ventilador de techo/Pintar habitación/Reparar fuga]
Experiencia: [Mi nivel]
Herramientas disponibles: [Lista]

Incluye:
- Precauciones de seguridad
- Herramientas necesarias
- Materiales con cantidades
- Pasos detallados
- Errores comunes
- Resolución de problemas
```

## Selección de Materiales

```
Prompt: Ayúdame a elegir:

Proyecto: [Pisos/Pintura/Accesorios]
Habitación: [Descripción]
Estilo: [Moderno/Tradicional/etc]
Presupuesto: [Por pie cuadrado]
Uso: [Tráfico alto/bajo]

Recomienda opciones con pros/contras.
```

## Estimación de Costos

```
Prompt: Estima costos:

Proyecto: [Descripción]
Ubicación: [Ciudad, Estado]
Tamaño: [Pies cuadrados/unidades]
Calidad: [Económico/Medio/Alto nivel]

Desglose:
- Materiales
- Mano de obra (si contratas)
- Permisos
- Contingencia
- Rango total
```

## Visión de Antes y Después

```
Prompt: Describe transformación:

Actual: [Descripción de habitación]
Objetivos: [Lo que quiero]
Presupuesto: [Cantidad]
Estilo: [Preferencias]

Sugiere:
- Victorias rápidas
- Cambios mayores
- Enfoque por fases
- Resultado esperado
```

## Resolución de Problemas

```
Prompt: Diagnostica problema:

Problema: [Grifo con fuga/Luz parpadeante/etc]
Observaciones: [Lo que ves/escuchas]
Cuándo ocurre: [Momento]
Lo que has intentado: [Intentos]

Proporciona:
- Causas probables
- Arreglos de bricolaje
- Cuándo llamar a profesional
- Preocupaciones de seguridad
```
