# IA para Relaciones y Habilidades Sociales

## Ayuda con Comunicación

**Conversaciones difíciles**:
```
Necesito hablar con mi compañero de cuarto sobre:
- A menudo llega tarde pagando el alquiler
- Me está causando estrés
- No quiero dañar la amistad

¿Cómo debería abordar esta conversación? Dame:
- Líneas de apertura
- Puntos clave a cubrir
- Cómo mantenerme calmado y constructivo
```

**Escucha activa**:
```
Enséñame técnicas de escucha activa para cuando mi pareja está molesta. Incluye:
- Frases específicas a usar
- Consejos de lenguaje corporal
- Qué evitar decir
- Cómo validar sentimientos sin intentar "arreglar" todo
```

## Resolución de Conflictos

**Enfriamiento de discusiones**:
```
Mi pareja y yo discutimos sobre [quehaceres domésticos/finanzas/etc.]. Ambos dijimos cosas que lamentamos.

¿Cómo puedo:
1. Enfriarme efectivamente
2. Abordar la reconciliación
3. Tener una conversación de seguimiento productiva
4. Prevenir este patrón
```

**Creación de disculpas**:
```
Necesito disculparme con mi amigo por:
- Cancelar planes a último minuto
- Tercera vez este mes
- Están molestos y entiendo por qué

Ayúdame a escribir una disculpa sincera que:
- Reconozca el impacto
- No ponga excusas
- Muestre que haré mejor
```

## Citas y Relaciones

**Ideas de primera cita**:
```
Sugiere 5 ideas creativas de primera cita que:
- No sean cena/película (demasiado común)
- Permitan conversación
- Presupuesto: $50 o menos
- Escenario: Área urbana
- Muestren consideración
```

**Iniciadores de conversación**:
```
Dame 10 temas de conversación interesantes para primera cita que:
- Vayan más allá de charla trivial
- No sean demasiado personales/pesados
- Revelen personalidad
- Mantengan las cosas divertidas y atractivas
```

**Chequeos de relación**:
```
Crea un marco de chequeo mensual de relación para parejas:
- Preguntas importantes para hacerse mutuamente
- Temas a revisar (metas, satisfacción, preocupaciones)
- Cómo hacerlo cómodo, no interrogatorio
- Mantenerlo bajo 30 minutos
```

## Dinámicas Familiares

**Establecimiento de límites**:
```
Ayúdame a establecer límites con mis padres que:
- Llegan sin avisar
- Los amo pero necesito más aviso
- Quiero ser respetuoso pero firme

Proporciona:
- Cómo explicar mi necesidad
- Declaraciones de límites específicas
- Cómo manejar objeciones
```

**Negociaciones de días festivos**:
```
Mi pareja y yo tenemos padres divorciados (4 hogares en total). Ayúdanos a planear visitas justas de días festivos:
- Se acercan Acción de Gracias y Navidad
- Queremos ser justos con todos
- También necesitamos tiempo para nosotros
- Vivimos a 2 horas de la mayoría de las familias

Crea una estrategia para discutir esto con las familias.
```

## Amistad

**Hacer nuevos amigos siendo adulto**:
```
Me mudé a una nueva ciudad y quiero hacer amigos. Soy:
- Introvertido pero social
- Intereses: Senderismo, juegos de mesa, cocina
- Principios de los 30
- Trabajo desde casa (así que no hay amigos del trabajo)

¿Dónde y cómo debería conocer personas? Da pasos específicos y accionables.
```

**Mantener amistades a larga distancia**:
```
Mi mejor amigo se mudó al otro lado del país. ¿Cómo podemos mantenernos cerca? Sugiere:
- Actividades regulares que podamos hacer remotamente
- Formas creativas de compartir la vida
- Frecuencia de contacto que sea realista
- Gestos especiales para mostrar que me importa
```

**Manejar rupturas de amistades**:
```
Una amistad de mucho tiempo se está desvaneciendo naturalmente. Hemos crecido en diferentes direcciones. ¿Está bien dejarla desvanecer, o debería abordarlo activamente?

Si abordar: ¿Cómo hacerlo con gracia?
Si dejar desvanecer: ¿Cómo lidiar con la pérdida?
```

## Situaciones Sociales

**Dominio de charla trivial**:
```
Tengo dificultades con charla trivial en eventos de networking. Dame:
- 10 iniciadores de conversación versátiles
- Cómo salir con gracia de una conversación
- Temas a evitar
- Cómo recordar nombres
- Hacer seguimiento después del evento
```

**Declinar invitaciones**:
```
Necesito decir no a eventos sociales sin ofender. Crea plantillas para:
1. Declinar una boda (demasiado caro viajar)
2. Decir no a bebidas después del trabajo (necesito límites)
3. Omitir la fiesta de un amigo (solo necesito tiempo de inactividad)

Cada una debería ser: Honesta pero amable, firme pero cálida
```

**Consejos de anfitrión**:
```
Estoy organizando una cena para 8 personas (mezcla de amigos y conocidos).

Ayuda con:
- Planificación de menú (incluidas restricciones dietéticas)
- Facilitación de conversación
- Cronometraje de la velada
- Hacer que todos se sientan cómodos
- Qué preparar con anticipación
```

## Relaciones Laborales

**Pedir un aumento**:
```
Quiero pedir un aumento:
- He estado aquí 2 años
- Asumí más responsabilidad
- La tarifa de mercado es 15% más alta
- Buenas evaluaciones de desempeño

Ayúdame a preparar:
- Cuándo y cómo preguntar
- Cómo presentar mi caso
- Qué decir y qué no decir
- Cómo manejar diferentes respuestas
```

**Lidiar con colegas difíciles**:
```
Un colega constantemente:
- Se atribuye el mérito de mis ideas
- Me interrumpe en reuniones
- Me socava sutilmente

Quiero manejarlo profesionalmente. ¿Qué debería:
1. Hacer en el momento
2. Documentar
3. Discutir con ellos directamente
4. Escalar si es necesario
```

## Automejoramiento

**Inteligencia emocional**:
```
Me han dicho que no soy bueno leyendo emociones. ¿Cómo puedo mejorar la inteligencia emocional?

Proporciona:
- Ejercicios prácticos diarios
- Qué observar en otros
- Cómo verificar mis interpretaciones
- Recursos para aprender más
```

**Construcción de empatía**:
```
Ayúdame a practicar empatía en conflictos. Cuando alguien está molesto, guíame a través de:
1. Qué notar
2. Qué decir
3. Qué NO decir
4. Cómo validar sin estar de acuerdo
5. Errores comunes a evitar
```

## Sensibilidad Cultural

**Comunicación intercultural**:
```
Estoy trabajando con un equipo de [país/cultura]. ¿Qué debería saber sobre:
- Estilos de comunicación
- Malentendidos comunes
- Saludos apropiados
- Etiqueta de dar regalos
- Normas de reuniones de negocios
```

## Relaciones de Crianza

**Comunicación de co-crianza**:
```
Mi ex pareja y yo necesitamos co-criar mejor. Crea un marco de comunicación:
- Qué discutir (solo sobre niños vs. otros temas)
- Cómo compartir información eficientemente
- Manejar desacuerdos sobre niños
- Mantenerlo civil y enfocado
```

**Hablar con niños sobre temas difíciles**:
```
Necesito explicar [divorcio/muerte/mudanza/etc.] a mi hijo de 8 años.

Ayúdame a:
- Usar lenguaje apropiado para la edad
- Ser honesto pero no aterrador
- Responder preguntas probables
- Proporcionar seguridad
- Saber qué evitar decir
```

## Dar Regalos

**Ideas de regalos reflexivos**:
```
Necesito un regalo significativo para:
- Relación: [Mamá/mejor amigo/pareja/etc.]
- Ocasión: [Cumpleaños/aniversario/solo porque]
- Sus intereses: [lista]
- Presupuesto: $[cantidad]
- Meta: Mostrar que presto atención y me importa

Sugiere 5 opciones reflexivas con ideas de personalización.
```

## Gratitud y Apreciación

**Notas de agradecimiento**:
```
Escribe una nota de agradecimiento sincera para:
- Qué: Alguien me ayudó durante un momento difícil
- Cómo ayudaron: [acciones específicas]
- Impacto: Hizo una diferencia real
- Tono: Genuino, cálido, no exagerado
```

---

**Nota**: La IA puede proporcionar perspectivas y sugerencias, pero problemas de relaciones complejos pueden beneficiarse de asesoramiento profesional. Estos prompts son para orientación general y reflexión.
