# IA para Búsqueda de Empleo

## Optimización de Currículo

### Adaptación a Descripción de Puesto

```
Prompt: Optimiza mi currículo para este trabajo:

Mi currículo: [pega currículo]

Descripción del puesto: [pega descripción]

Adapta mi:
- Resumen
- Sección de habilidades
- Descripciones de experiencia
- Palabras clave para ATS

Mantén la veracidad.
```

### Formato Compatible con ATS

```
Prompt: Revisa mi currículo para compatibilidad con ATS:

[pega currículo]

Verifica:
- Problemas de análisis
- Optimización de palabras clave
- Problemas de formato
- Secciones faltantes
```

## Generación de Carta de Presentación

```
Prompt: Escribe una carta de presentación:

Trabajo: [título] en [empresa]
Requisitos clave: [de descripción]
Mi experiencia relevante:
- [Experiencia 1]
- [Experiencia 2]
- [Experiencia 3]

Tono: Profesional pero personalizable
Longitud: 250-300 palabras
Destacar: por qué soy ideal
```

## Optimización de LinkedIn

### Resumen de Perfil
```
Prompt: Escribe resumen de LinkedIn:

Rol: [Tu título]
Experiencia: [Años]
Especialidades: [Tus habilidades]
Logros: [Victorias clave]
Buscando: [Tipo de trabajo]

Hazlo: Atractivo, rico en palabras clave, 150 palabras
```

### Ideas de Publicaciones
```
Prompt: Genera 5 ideas de publicaciones de LinkedIn sobre:
- Búsqueda de empleo
- [Tu industria]
- Desarrollo profesional

Incluye: Gancho, valor, llamado a la acción
```

## Preparación para Entrevistas

### Investigación de Empresa
```
Prompt: Resumen de investigación para entrevista:

Empresa: [Nombre de empresa]
Industria: [Industria]
Rol: [Posición]

Proporciona:
- Resumen de empresa
- Noticias recientes
- Competidores
- Perspectivas de cultura
- Preguntas para hacer
```

### Preguntas Comunes
```
Prompt: Ayúdame a preparar respuestas para:

1. "Háblame de ti"
2. "¿Por qué este rol?"
3. "Mayor fortaleza/debilidad"
4. "¿Dónde te ves en 5 años?"

Basado en:
- Rol: [Título del trabajo]
- Mi trasfondo: [Resumen]
- Empresa: [Nombre de empresa]

Usa método STAR para ejemplos.
```

### Entrevista Simulada
```
Prompt: Realiza entrevista simulada:

Rol: [Posición]
Industria: [Industria]
Experiencia: [Mi nivel]

Hazme 5 preguntas de entrevista una a la vez.
Después de cada respuesta, proporciona retroalimentación sobre:
- Fortalezas
- Áreas para mejorar
- Revisiones sugeridas
```

## Negociación Salarial

```
Prompt: Ayúdame a negociar salario:

Oferta: $[cantidad]
Tasa de mercado: $[rango]
Mi experiencia: [Años]
Ubicación: [Ciudad]
Beneficios: [Oferta actual]

Proporciona:
- Sugerencia de contraoferta
- Guión de negociación
- Solicitudes alternativas si salario fijo
- Plantilla de email
```

## Mensajes de Networking

### Conexión de LinkedIn
```
Prompt: Escribe mensaje de conexión de LinkedIn:

Para: [Nombre], [Título] en [Empresa]
Contexto: [Cómo los encontraste]
Terreno común: [Interés/experiencia compartida]
Objetivo: [Lo que esperas]

Longitud: Menos de 300 caracteres
Tono: Profesional, genuino
```

### Entrevista Informativa
```
Prompt: Solicita entrevista informativa:

Para: [Nombre]
Sobre: [Su rol/empresa]
Por qué: [Tu interés]
Pedir: Llamada de 15-20 minutos

Hazlo fácil de aceptar.
```

## Seguimiento de Aplicaciones

### Sistema de Organización
```
Prompt: Crea sistema de seguimiento de búsqueda de empleo:

Rastrear:
- Empresa
- Posición
- Fecha de aplicación
- Estado
- Fechas de seguimiento
- Contactos
- Notas

Formato: Estructura de hoja de cálculo
```

## Plantillas de Email

### Email de Agradecimiento
```
Prompt: Escribe email de agradecimiento por entrevista:

Entrevistador: [Nombre]
Posición: [Rol]
Fecha de entrevista: [Fecha]
Puntos clave de discusión:
- [Punto 1]
- [Punto 2]

Incluye:
- Gratitud
- Reiterar interés
- Referenciar conversación específica
- Siguientes pasos

Longitud: 150-200 palabras
```

### Seguimiento
```
Prompt: Escribe email de seguimiento:

Situación: Apliqué hace [X] semanas, sin respuesta
Empresa: [Nombre de empresa]
Posición: [Rol]

Objetivo: Recordatorio gentil
Tono: Profesional, no insistente
Longitud: Menos de 100 palabras
```

## Análisis de Descripción de Puesto

```
Prompt: Analiza esta descripción de puesto:

[Pega descripción]

Extrae:
- Habilidades imprescindibles
- Habilidades deseables
- Indicadores de cultura de empresa
- Señales de alerta
- Preguntas para hacer
- Estimación de rango salarial
```

## Portafolio/Proyectos

```
Prompt: Describe este proyecto para mi portafolio:

Proyecto: [Nombre]
Qué hace: [Descripción]
Tecnologías: [Lista]
Mi rol: [Tu contribución]
Resultado: [Resultados]

Escribe:
- Título atractivo
- Descripción de 2 párrafos
- Logros clave
- Detalles técnicos

Audiencia: Gerentes de contratación
```

## Mejores Prácticas

✅ **Hacer**:
- Personalizar salida de IA
- Verificar todo
- Mantener autenticidad
- Usar como punto de partida
- Practicar respuestas en voz alta

❌ **No hacer**:
- Copiar-pegar sin editar
- Fabricar experiencia
- Usar aplicaciones genéricas
- Depender solo de IA
- Omitir corrección
