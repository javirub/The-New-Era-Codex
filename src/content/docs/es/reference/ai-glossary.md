# Glosario de IA y LLM

## Conceptos Básicos de IA

**Inteligencia Artificial (IA)**: Sistemas que simulan la inteligencia humana

**Aprendizaje Automático (Machine Learning - ML)**: Sistemas de IA que aprenden de datos

**Aprendizaje Profundo (Deep Learning)**: ML usando redes neuronales con múltiples capas

**Red Neuronal (Neural Network)**: Sistemas de computación inspirados en cerebros biológicos

**Entrenamiento (Training)**: Proceso de enseñar a un modelo de IA usando datos

**Inferencia (Inference)**: Usar un modelo entrenado para hacer predicciones

**Ajuste Fino (Fine-Tuning)**: Adaptar un modelo pre-entrenado a tareas específicas

## Términos Específicos de LLM

**Modelo de Lenguaje Grande (Large Language Model - LLM)**: IA entrenada en texto vasto para entender y generar lenguaje

**Transformer**: Arquitectura de red neuronal que impulsa los LLM modernos

**Token**: Unidad básica de texto (palabra, subpalabra o carácter)

**Ventana de Contexto (Context Window)**: Máximo de tokens que un LLM puede procesar a la vez

**Temperatura (Temperature)**: Controla la aleatoriedad (0=determinístico, 2=creativo)

**Top-P (Nucleus Sampling)**: Método de muestreo para generación

**Prompt**: Texto de entrada dado a un LLM

**Completion**: Respuesta generada por el LLM

**Prompt de Sistema (System Prompt)**: Instrucciones que definen el comportamiento del LLM

## Conceptos Avanzados

**RAG (Generación Aumentada por Recuperación)**: Mejorar LLMs con recuperación de conocimiento externo

**Embedding**: Representación numérica de texto

**Base de Datos Vectorial (Vector Database)**: Almacena embeddings para búsqueda por similitud

**Búsqueda Semántica (Semantic Search)**: Encontrar significado, no solo palabras clave

**Alucinación (Hallucination)**: Cuando el LLM genera información falsa

**Aprendizaje de Pocos Ejemplos (Few-Shot Learning)**: Aprender de pocos ejemplos

**Aprendizaje Sin Ejemplos (Zero-Shot Learning)**: Realizar tareas sin ejemplos

**Cadena de Pensamiento (Chain-of-Thought)**: Enfoque de razonamiento paso a paso

**Agente (Agent)**: Sistema de IA que puede tomar acciones

**Uso de Herramientas (Tool Use)**: LLM llamando funciones/APIs externas

## Componentes del Modelo

**Parámetros (Parameters)**: Valores aprendidos en la red neuronal

**Capas (Layers)**: Niveles de procesamiento en la red neuronal

**Mecanismo de Atención (Attention Mechanism)**: Cómo los modelos se enfocan en información relevante

**Tokenizador (Tokenizer)**: Convierte texto en tokens

**Vocabulario (Vocabulary)**: Todos los tokens que un modelo conoce

**Estado Oculto (Hidden State)**: Representaciones internas del modelo

## Conceptos de Entrenamiento

**Pre-entrenamiento (Pre-training)**: Entrenamiento inicial en grandes conjuntos de datos

**Aprendizaje Supervisado (Supervised Learning)**: Entrenamiento con datos etiquetados

**Aprendizaje No Supervisado (Unsupervised Learning)**: Entrenamiento con datos sin etiquetar

**Aprendizaje por Refuerzo (Reinforcement Learning)**: Aprender a través de recompensas

**RLHF (Aprendizaje por Refuerzo desde Retroalimentación Humana)**: Entrenamiento usando preferencias humanas

**Sobreajuste (Overfitting)**: El modelo memoriza los datos de entrenamiento

**Subajuste (Underfitting)**: Modelo demasiado simple para aprender patrones

**Generalización (Generalization)**: Desempeño bueno en datos nuevos

## Métricas de Evaluación

**Perplexity (Perplejidad)**: Qué tan bien el modelo predice texto

**BLEU**: Calidad de traducción automática

**ROUGE**: Calidad de resumen

**F1 Score (Puntuación F1)**: Rendimiento de clasificación

**Accuracy (Precisión)**: Porcentaje de predicciones correctas

**Precision (Precisión)**: Verdaderos positivos / positivos predichos

**Recall (Recuperación)**: Verdaderos positivos / positivos reales

## API y Despliegue

**API (Interfaz de Programación de Aplicaciones)**: Interfaz para interacción de software

**Endpoint**: Punto de acceso a la API

**Latency (Latencia)**: Tiempo de respuesta

**Throughput (Rendimiento)**: Solicitudes manejadas por unidad de tiempo

**Rate Limit (Límite de Tasa)**: Máximo de solicitudes permitidas

**Batch Processing (Procesamiento por Lotes)**: Procesar múltiples elementos juntos

**Streaming (Transmisión)**: Salida de tokens en tiempo real

**Caching (Almacenamiento en Caché)**: Almacenar resultados para reutilización

## Términos Especializados

**Multimodal**: Procesar múltiples tipos de datos (texto, imagen, audio)

**Vision Model (Modelo de Visión)**: Procesa imágenes

**Whisper**: Reconocimiento de voz de OpenAI

**DALL-E**: Generación de imágenes de OpenAI

**GPT**: Transformador Generativo Pre-entrenado (Generative Pre-trained Transformer)

**BERT**: Representaciones de Codificador Bidireccional de Transformers (Bidirectional Encoder Representations from Transformers)

**LoRA (Adaptación de Bajo Rango)**: Método eficiente de ajuste fino

**Quantization (Cuantización)**: Reducir el tamaño/precisión del modelo

**Distillation (Destilación)**: Crear un modelo más pequeño a partir de uno más grande

## Ética y Seguridad

**Bias (Sesgo)**: Preferencias injustas en las salidas del modelo

**Fairness (Equidad)**: Trato equitativo entre grupos

**Alignment (Alineación)**: Hacer coincidir el comportamiento de la IA con valores humanos

**Safety (Seguridad)**: Prevenir salidas dañinas

**Red Teaming (Equipo Rojo)**: Probar vulnerabilidades

**Content Policy (Política de Contenido)**: Reglas para salidas permitidas

**Moderation (Moderación)**: Filtrar contenido inapropiado

## Términos de Negocios

**Token Cost (Costo por Token)**: Precio por token usado

**Context Length (Longitud de Contexto)**: Historial máximo de conversación

**Fine-Tuning Cost (Costo de Ajuste Fino)**: Precio para personalizar modelos

**Inference Cost (Costo de Inferencia)**: Costo por llamada a API

**ROI (Retorno de Inversión)**: Valor ganado vs costo

**SLA (Acuerdo de Nivel de Servicio)**: Tiempo de actividad/rendimiento garantizado
