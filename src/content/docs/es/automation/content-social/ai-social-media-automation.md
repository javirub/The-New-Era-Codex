# Automatización de Redes Sociales con IA

## Flujo de Trabajo de Generación de Contenido

```
1. Generar ideas de posts (ChatGPT)
2. Crear visuales (DALL-E/Canva)
3. Optimizar captions (herramientas IA)
4. Programar posts (Buffer/Hootsuite)
5. Monitorear engagement (análisis IA)
6. Responder automáticamente (chatbots)
```

## Stack de Herramientas

**Creación de Contenido**:
- ChatGPT: Redacción de posts
- DALL-E/Midjourney: Imágenes
- Canva AI: Plantillas de diseño
- Lately: Reutilizar contenido largo

**Programación**:
- Buffer: Programación multi-plataforma
- Hootsuite: Gestión empresarial
- Later: Planificación visual

**Análisis**:
- Sprout Social: Insights con IA
- Brandwatch: Escucha social

## Ejemplos de Prompts

**Generación de posts**:
```
Create 10 LinkedIn posts about [topic]:
- Hook-driven opens
- Value-focused content
- Clear CTAs
- Mix: tips, stories, questions
- 150-200 words each
```

**Investigación de hashtags**:
```
Suggest hashtags for [content]:
- 3 high-traffic hashtags
- 5 niche-specific hashtags
- 2 branded hashtags
Explain strategy for each
```

**Respuestas de engagement**:
```
Draft responses to common comments:
1. "Tell me more"
2. "Thanks for sharing"
3. "I disagree because..."

Maintain brand voice: [description]
```

## Flujo de Trabajo de Automatización (Zapier)

```
1. Trigger: RSS feed new post
2. AI: Generate social post from article
3. AI: Create image prompt
4. DALL-E: Generate image
5. Buffer: Schedule post
6. Slack: Notify team
```
