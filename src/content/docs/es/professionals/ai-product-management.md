# IA para Gestión de Productos

## Estrategia de Producto

```
Prompt: Analiza estrategia de producto:

Producto: [Nombre]
Mercado: [Mercado objetivo]
Competidores: [Lista]
Estado actual: [Descripción]
Objetivos: [Objetivos]

Proporciona:
- Análisis de posicionamiento de mercado
- Oportunidades de diferenciación
- Recomendaciones estratégicas
- Riesgos
- Métricas de éxito
```

## Análisis de Investigación de Usuarios

```
Prompt: Analiza retroalimentación de usuarios:

Retroalimentación: [Notas de entrevistas/respuestas de encuestas]

Extrae:
- Puntos de dolor comunes
- Solicitudes de funciones
- Patrones de personas de usuario
- Temas priorizados
- Perspectivas accionables
- Citas destacadas
```

## Priorización de Funciones

```
Prompt: Ayuda a priorizar funciones:

Funciones: [Lista con descripciones]

Criterios:
- Impacto en usuario (1-10)
- Valor de negocio (1-10)
- Esfuerzo de implementación (1-10)
- Alineación estratégica (1-10)

Clasifica usando RICE o marco similar.
Explica razonamiento.
```

## Requisitos de Producto

```
Prompt: Escribe PRD para:

Función: [Nombre de función]
Problema: [Problema del usuario]
Objetivos: [Lo que logra]
Usuarios: [Usuarios objetivo]
Métricas de éxito: [KPIs]

Incluye:
- Historias de usuario
- Criterios de aceptación
- Casos extremos
- Dependencias
- Preguntas abiertas
```

## Planificación de Hoja de Ruta

```
Prompt: Crea hoja de ruta de producto:

Cronograma: [Próximos 6-12 meses]
Temas: [Temas estratégicos]
Recursos: [Tamaño de equipo]
Restricciones: [Técnicas/de negocio]

Formato:
- Ahora/Próximo/Después
- Iniciativas clave por trimestre
- Dependencias
- Métricas de éxito
```

## Análisis Competitivo

```
Prompt: Análisis competitivo:

Nuestro producto: [Descripción]
Competidores: [Lista]
Funciones a comparar: [Lista]

Crea:
- Matriz de comparación de funciones
- Fortalezas/debilidades
- Brechas de mercado
- Oportunidades
- Evaluación de amenazas
```

## Comunicaciones de Lanzamiento

```
Prompt: Escribe notas de lanzamiento:

Versión: [Número]
Audiencia: [Usuarios/Técnico]
Cambios: [Lista]

Formato:
- Novedades
- Mejoras
- Correcciones de errores
- Problemas conocidos
- Guía de migración (si es necesario)

Tono: [Claro, emocionante, útil]
```
