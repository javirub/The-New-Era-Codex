# IA para Gestión de Proyectos

## Aplicaciones de IA en GP

### Planificación
- Desglose de proyecto
- Estimación de cronogramas
- Asignación de recursos
- Mapeo de dependencias

### Ejecución
- Actualizaciones de estado
- Monitoreo de riesgos
- Resúmenes de reuniones
- Seguimiento de elementos de acción

### Análisis
- Métricas de desempeño
- Analítica predictiva
- Perspectivas retrospectivas
- Recomendaciones de mejora

## Herramientas de IA para GP

- **Motion**: Programación con IA
- **Trevor**: Gestión inteligente de tareas
- **Reclaim**: Optimización de calendario
- **Asana Intelligence**: Flujos de trabajo inteligentes

## Planificación de Proyectos con IA

```
Prompt: "Desglosa este proyecto:

Objetivo: [Objetivo del proyecto]
Cronograma: [Duración]
Equipo: [Tamaño y roles]
Restricciones: [Presupuesto, recursos]

Proporciona:
- Estructura de desglose de trabajo (EDT)
- Hitos clave con fechas
- Ruta crítica
- Asignación de recursos
- Factores de riesgo
- Métricas de éxito"
```

## Evaluación de Riesgos

```
Prompt: "Analiza riesgos del proyecto:

Detalles del proyecto: [Descripción]
Experiencia del equipo: [Nivel]
Dependencias: [Lista]
Cronograma: [Duración]

Para cada riesgo:
- Probabilidad (A/M/B)
- Impacto (A/M/B)
- Estrategia de mitigación
- Plan de contingencia"
```

## Eficiencia de Reuniones

**Preparación de reunión con IA**:
```
Prompt: "Prepara para reunión de proyecto:
- Revisa notas de última reunión: [notas]
- Estado de sprint actual: [estado]
- Decisiones clave necesarias: [lista]

Crea:
- Agenda
- Preguntas de discusión
- Marco de decisión"
```

**Procesamiento post-reunión**:
```
Prompt: "Procesa notas de reunión:
[pega transcripción/notas]

Extrae:
- Decisiones tomadas
- Elementos de acción (responsable, fecha de entrega)
- Elementos de estacionamiento
- Elementos de agenda para próxima reunión"
```

## Mejores Prácticas

✅ **Hacer**:
- Usar IA para planificación inicial
- Validar estimaciones con equipo
- Actualizar contexto de IA regularmente
- Combinar IA con herramientas de GP
- Revisar decisiones críticas humanamente

❌ **No hacer**:
- Seguir ciegamente cronogramas de IA
- Omitir aporte del equipo
- Ignorar contexto
- Automatizar excesivamente comunicación
