# IA para Investigación y Análisis Legal

## Revisión de Contratos

**Extracción de términos clave**:
```
Revisa este contrato y extrae:

[pega sección de contrato o contrato completo]

Identifica:
- Obligaciones clave para cada parte
- Términos de pago
- Cláusulas de terminación
- Limitaciones de responsabilidad
- Provisiones inusuales o preocupantes
- Protecciones estándar faltantes
```

**Evaluación de riesgo**:
```
Analiza este contrato de proveedor para riesgos:

[pega contrato]

Enfócate en:
- Exposición financiera
- Límites de responsabilidad (¿adecuados o demasiado bajos?)
- Indemnización (¿unilateral?)
- Cláusulas de auto-renovación
- Provisiones de privacidad de datos
- Resolución de disputas (¿lugar favorable?)

Califica cada riesgo: Alto/Medio/Bajo
```

**Sugerencias de redlining**:
```
Necesito negociar esta cláusula de contrato:

Lenguaje actual:
"[pega cláusula]"

Problemas con lenguaje actual:
- [tus preocupaciones]

Sugiere:
- Lenguaje alternativo que proteja mis intereses
- Redacción estándar de la industria
- Qué rechazar vs. aceptar
```

## Investigación Legal

**Investigación de jurisprudencia**:
```
Necesito precedentes para:

Cuestión: [pregunta legal]
Jurisdicción: [estatal/federal]
Área de práctica: [área del derecho]

Ayúdame a:
- Enmarcar la pregunta legal
- Identificar jurisprudencia relevante
- Resumir holdings clave
- Distinguir casos desfavorables
- Encontrar argumentos de apoyo
```

**Interpretación estatutaria**:
```
Interpreta este estatuto:

[pega texto del estatuto]

Contexto: [situación a la que lo estás aplicando]

Explica:
- Significado en lenguaje simple
- Definiciones de términos clave
- Excepciones o exclusiones
- Cómo se aplica a mi situación
- Ambigüedades potenciales
```

## Escritura Legal

**Carta de demanda**:
```
Redacta una carta de demanda para:

Situación: [describe disputa]
Destinatario: [parte]
Lo que quieres: [remedio]
Tono: [Firme pero profesional / amenazando litigio / orientado a acuerdo]

Incluye:
- Declaración de hechos
- Base legal para reclamo
- Demanda/remedio
- Fecha límite para respuesta
- Consecuencias si se ignora
```

**Borrador de moción para desestimar**:
```
Redacta argumentos para moción para desestimar basada en:

Fundamentos: [falta de jurisdicción / falta de causa de acción / prescripción]
Hechos: [hechos relevantes]
Ley aplicable: [jurisdicción]

Estructura: Introducción, Estándar de Revisión, Argumento, Conclusión
```

## Debida Diligencia

**Lista de verificación de M&A**:
```
Crea una lista de verificación de debida diligencia legal para adquirir:

Objetivo: [tipo de empresa]
Industria: [industria]
Tamaño: [ingresos/empleados]
Jurisdicción: [ubicación]

Categorías:
- Estructura corporativa
- Contratos y acuerdos
- Propiedad intelectual
- Litigio y cumplimiento
- Asuntos de empleo
- Bienes raíces y activos
```

**Auditoría de cumplimiento**:
```
Nuestra empresa necesita cumplir con [regulación - GDPR/HIPAA/SOX/etc.].

Crea una lista de verificación de auditoría:
- Qué necesitamos revisar
- Documentos a reunir
- Procesos a examinar
- Violaciones comunes a verificar
- Pasos de remediación si se encuentran brechas
```

## Plantillas de Contratos

**Plantilla de NDA**:
```
Crea una plantilla de NDA mutua para:

Partes: Dos empresas explorando asociación
Alcance: Tecnología y estrategia de negocio
Duración: 2 años
Jurisdicción: [estado]

Incluye:
- Definición de información confidencial
- Usos permitidos
- Exclusiones de confidencialidad
- Devolución/destrucción de materiales
- Remedios por incumplimiento
```

**Bosquejo de acuerdo de servicio**:
```
Bosqueja un acuerdo de servicio para:

Proveedor de servicio: [tipo]
Cliente: [tipo]
Servicios: [descripción]
Pago: [estructura]

Secciones clave necesarias:
- Alcance de trabajo
- Entregables y cronograma
- Términos de pago
- Propiedad de PI
- Garantías y descargos
- Derechos de terminación
- Limitaciones de responsabilidad
```

## Cumplimiento Regulatorio

**Análisis de brechas de cumplimiento**:
```
Estamos lanzando [producto/servicio] en [jurisdicción].

Regulaciones relevantes: [lista]

Ayúdame a:
- Identificar todos los requisitos aplicables
- Crear lista de verificación de cumplimiento
- Resaltar áreas de incertidumbre
- Sugerir procedimientos de cumplimiento
- Identificar cuándo consultar abogado especialista
```

**Generador de política de privacidad**:
```
Crea un bosquejo de política de privacidad para:

Negocio: [tipo]
Datos recopilados: [tipos]
Cómo se usan: [propósitos]
Terceros: [si hay]
Derechos del usuario: [acceso, eliminación, etc.]
Jurisdicción: [ubicación]

Cumpliendo con: GDPR, CCPA, otras leyes aplicables
```

## Resolución de Disputas

**Estrategia de negociación**:
```
Estoy negociando acuerdo de:

Disputa: [descripción]
Nuestra posición: [reclamo/defensa]
Su posición: [reclamo opuesto]
Nuestro mejor resultado: [resultado ideal]
Nuestro BATNA: [alternativa si no hay acuerdo]

Crea:
- Posición de apertura
- Posiciones de respaldo
- Estrategia de concesión
- Rompedores de trato
- Soluciones creativas a explorar
```

**Preparación de mediación**:
```
Prepárame para mediación:

Disputa: [cuestión]
Nuestras fortalezas: [lista]
Nuestras debilidades: [lista]
Argumentos probables de otra parte: [lista]

Crea:
- Bosquejo de declaración de apertura
- Puntos clave a enfatizar
- Desafíos anticipados y respuestas
- Parámetros de autoridad de acuerdo
```

## Propiedad Intelectual

**Estrategia de búsqueda de marca**:
```
Quiero registrar marca "[marca propuesta]" para [bienes/servicios].

Ayúdame a:
- Evaluar probabilidad de registro
- Estrategia de búsqueda de marcas confusamente similares
- Identificar conflictos potenciales
- Sugerir variaciones si existen conflictos
- Entender alcance de protección
```

**Análisis de copyright**:
```
¿Es esto uso justo?

Uso: [cómo quieres usar material con copyright]
Trabajo original: [descripción]
Propósito: [comercial/educativo/comentario]
Cantidad usada: [porción]
Efecto en mercado: [impacto]

Analiza bajo factores de uso justo. ¿Nivel de riesgo?
```

**Búsqueda de arte previo de patente**:
```
Invención: [descripción]
Características clave: [elementos novedosos]

Ayúdame a:
- Enmarcar consulta de búsqueda
- Identificar clases de patente relevantes
- Términos clave a buscar
- Cómo evaluar resultados
- Qué hace relevante el arte previo
```

## Derecho Laboral

**Políticas de manual de empleado**:
```
Redacta una política para:

Tema: [Trabajo remoto / Redes sociales / Reembolso de gastos / etc.]
Tipo de empresa: [industria, tamaño]
Jurisdicción: [estado]

Incluye:
- Declaración clara de política
- Procedimientos
- Responsabilidades del empleado
- Consecuencias por violaciones
- Cumplimiento con leyes aplicables
```

**Lista de verificación de terminación**:
```
Necesitamos terminar a un empleado:

Razón: [desempeño / despido / mala conducta]
Estado del empleado: [a voluntad / contrato / consideraciones de clase protegida]
Estado: [jurisdicción]

Lista de verificación:
- Requisitos legales
- Documentación necesaria
- Cálculos de pago final
- Continuación de beneficios
- Devolución de propiedad de empresa
- Pasos de mitigación de riesgo
```

## Gobierno Corporativo

**Plantilla de resolución de junta**:
```
Crea una resolución de junta para:

Acción: [lo que está aprobando la junta]
Empresa: [tipo]
Estado: [estado de incorporación]

Incluye:
- Cláusulas por cuanto (antecedentes)
- Cláusulas resueltas (acciones)
- Autoridad otorgada
- Bloques de firma
```

**Actas corporativas**:
```
Redacta actas de reunión para:

Tipo de reunión: [Junta / Accionistas / Comité]
Fecha: [fecha]
Asistentes: [lista]
Decisiones tomadas: [lista]

Formato: Actas corporativas formales incluyendo:
- Llamada al orden
- Aprobación de actas previas
- Informes
- Resoluciones
- Levantamiento
```

## Bienes Raíces

**Revisión de arrendamiento**:
```
Revisa este arrendamiento comercial:

[pega términos clave o arrendamiento completo]

Análisis de perspectiva de inquilino:
- Escalación de renta (¿razonable?)
- Responsabilidades de mantenimiento (¿asignación apropiada?)
- Restricciones de uso (¿demasiado limitantes?)
- Asignación/subarrendamiento (¿viable?)
- Derechos de terminación (¿adecuados?)
- Costos ocultos
```

**Problemas de acuerdo de compra**:
```
¿Qué debería negociar en este acuerdo de compra de bienes raíces?

[pega acuerdo o términos clave]

Buscando:
- Contingencias necesarias
- Período de debida diligencia
- Garantías del vendedor
- Asignación de riesgo
- Condiciones de cierre
- Términos desfavorables
```

## Apoyo de Litigio

**Solicitudes de descubrimiento**:
```
Redacta interrogatorios para:

Tipo de caso: [disputa contractual / lesiones personales / etc.]
Rol: [demandante / demandado]
Hechos clave en disputa: [lista]

Crea 15-20 interrogatorios dirigidos a:
- Establecer hechos
- Fijar su historia
- Identificar testigos/documentos
- Revelar debilidades
```

**Bosquejo de deposición**:
```
Voy a deponer a: [rol del testigo]
Caso: [breve descripción]
Qué necesito establecer: [metas]

Crea bosquejo de deposición:
- Preguntas de antecedentes
- Áreas de temas clave
- Oportunidades de impugnación
- Preguntas de revisión de documentos
- Preservación de testimonio para juicio
```

## Gestión de Riesgo

**Evaluación de riesgo legal**:
```
Evalúa riesgos legales de:

Actividad de negocio: [descripción]
Industria: [industria]
Jurisdicción: [ubicación]

Identifica:
- Riesgos regulatorios
- Riesgos contractuales
- Exposición de responsabilidad
- Riesgos de PI
- Problemas de derecho laboral
- Estrategias de mitigación de riesgo
```
