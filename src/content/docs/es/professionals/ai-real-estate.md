# IA para Bienes Raíces

## Descripciones de Propiedades

```
Prompt: Escribe listado de propiedad:

Detalles de propiedad:
- Tipo: [Casa/Condominio/etc]
- Habitaciones/Baños: [Números]
- Pies cuadrados: [Tamaño]
- Ubicación: [Vecindario]
- Características: [Lista aspectos destacados]
- Precio: $[cantidad]

Objetivo: [Compradores primerizos/Inversores/Lujo]
Tono: [Profesional/Cálido/Sofisticado]

Incluye:
- Titular convincente
- Características clave
- Aspectos destacados del vecindario
- Llamado a la acción
```

## Análisis de Mercado

```
Prompt: Analiza mercado inmobiliario:

Ubicación: [Área]
Tipo de propiedad: [Tipo]
Rango de precio: $[rango]
Período de tiempo: [Últimos X meses]

Datos: [Datos de mercado]

Proporciona:
- Tendencias de mercado
- Movimientos de precios
- Niveles de inventario
- Días en el mercado
- Análisis de mercado comprador/vendedor
- Pronóstico
```

## Análisis de Inversión

```
Prompt: Evalúa propiedad de inversión:

Propiedad: [Dirección/detalles]
Precio de compra: $[cantidad]
Potencial de renta: $[mensual]
Gastos: [Lista]
Pago inicial: [%]
Términos de préstamo: [Detalles]

Calcula:
- Flujo de efectivo
- Tasa CAP
- ROI
- Análisis de punto de equilibrio
- Riesgos
- Recomendación
```

## Plantillas de Email

```
Prompt: Escribe email de seguimiento:

Destinatario: [Comprador/Vendedor]
Contexto: [Visita/Oferta/etc]
Propiedad: [Dirección]
Siguientes pasos: [Lo que quieres]

Tono: Profesional pero personalizable
```

## Herramientas de IA para Bienes Raíces

- **ChatGPT**: Descripciones, análisis
- **Zillow**: Valuaciones de casas con IA
- **Redfin**: Perspectivas de mercado
- **Compass**: Plataforma con funciones de IA
