# IA para Cadena de Suministro y Operaciones

## Pronóstico de Demanda

```
Prompt: Pronostica demanda:

Producto: [Nombre del producto]
Datos históricos: [Datos de ventas]
Estacionalidad: [Patrones]
Factores externos: [Eventos/tendencias]
Cronograma: [Próximos 3/6/12 meses]

Proporciona:
- Pronóstico de demanda
- Intervalos de confianza
- Suposiciones clave
- Factores de riesgo
- Recomendaciones de inventario
```

## Optimización de Inventario

```
Prompt: Optimiza inventario:

Actual: [Niveles de stock]
Tiempos de entrega: [Días]
Demanda: [Pronóstico]
Costo de almacenamiento: $[cantidad]
Costo de desabasto: $[cantidad]

Recomienda:
- Puntos de reorden
- Niveles de stock de seguridad
- Cantidades de pedido
- Optimización de costos
```

## Análisis de Proveedores

```
Prompt: Evalúa proveedores:

Proveedores: [Lista con datos de desempeño]
Criterios:
- Calidad
- Precio
- Tiempo de entrega
- Confiabilidad
- Factores de riesgo

Clasifica y recomienda con razonamiento.
```

## Optimización de Rutas

```
Prompt: Optimiza rutas de entrega:

Ubicaciones: [Direcciones]
Restricciones:
- Capacidad de vehículo
- Ventanas de tiempo
- Horas de conductor
- Entregas prioritarias

Sugiere rutas óptimas con ahorro de tiempo/costo.
```
