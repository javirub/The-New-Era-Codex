# IA para Educadores

## Planificación de Lecciones

**Plan de lección completo**:
```
Crea un plan de lección para:

Nivel de grado: [grado]
Materia: [materia]
Tema: [tema específico]
Duración: [minutos]
Objetivos de aprendizaje: [qué deberían aprender los estudiantes]
Conocimiento previo: [qué ya saben los estudiantes]

Incluye:
- Actividad de gancho/compromiso (5-10 min)
- Bosquejo de instrucción directa
- Actividad de práctica guiada
- Práctica independiente
- Evaluación/verificación de comprensión
- Estrategias de diferenciación
- Materiales necesarios
```

**Objetivos de aprendizaje**:
```
Estoy enseñando [tema] a [nivel de grado].

Crea 3-5 objetivos de aprendizaje usando la Taxonomía de Bloom:
- 1-2 a nivel de recordar/comprender
- 2-3 a nivel de aplicar/analizar
- 1 a nivel de evaluar/crear

Formato: Los estudiantes serán capaces de [verbo de acción] [contenido] [condiciones/criterios]
```

**Estrategias de diferenciación**:
```
Mi clase tiene:
- 5 estudiantes avanzados
- 15 estudiantes a nivel de grado
- 3 estudiantes aprendiendo inglés
- 2 estudiantes con IEP (leyendo bajo nivel de grado)

Tema: [tema de lección]

Crea estrategias de diferenciación para:
- Contenido (qué aprenden)
- Proceso (cómo aprenden)
- Producto (cómo demuestran aprendizaje)

Específico a las necesidades de cada grupo.
```

## Compromiso Estudiantil

**Actividades de gancho**:
```
Crea 5 ganchos atractivos para:

Tema: [tema de lección]
Grado: [nivel de grado]
Duración: 5-10 minutos cada uno

Hazlos:
- Que capturen la atención
- Relevantes para la vida de los estudiantes
- Activen conocimiento previo
- Establezcan propósito para aprender

Variedad: Video, pregunta, demostración, historia, rompecabezas
```

**Ideas de gamificación**:
```
Gamifica el aprendizaje para:

Materia: [materia]
Tema: [tema]
Grado: [nivel de grado]

Crea:
- Sistema de puntos
- Niveles/insignias
- Desafíos de equipo
- Tabla de clasificación (sensible a todos los estudiantes)
- Estructura de recompensas

Mantenlo: Motivador, justo, fácil de gestionar
```

## Creación de Evaluaciones

**Generación de cuestionarios**:
```
Crea un cuestionario de 20 preguntas sobre:

Tema: [tema]
Nivel de grado: [grado]
Mezcla de formato:
- 10 de opción múltiple
- 5 verdadero/falso
- 5 respuesta corta

Distribución de dificultad:
- 50% a nivel de grado
- 30% ligeramente desafiante
- 20% avanzado

Incluye clave de respuestas con explicaciones.
```

**Creación de rúbrica**:
```
Crea una rúbrica para:

Tarea: [proyecto/ensayo/presentación]
Nivel de grado: [grado]

Criterios a evaluar:
- [Lista 4-6 criterios]

Formato: Escala de 4 puntos (Excede/Cumple/Acercándose/Debajo)

Para cada criterio y nivel:
- Descriptores claros
- Indicadores observables
- Ejemplos si es útil
```

**Preguntas de boleto de salida**:
```
Crea 5 preguntas de boleto de salida para:

Tema de lección: [lección de hoy]
Propósito: Verificar comprensión de [conceptos específicos]

Mezcla de:
- Evaluación rápida (¿qué aprendiste?)
- Autorreflexión (¿qué sigue confuso?)
- Aplicación (resolver un problema)

Cada una debería tomar 2-3 minutos para completar.
```

## Retroalimentación y Calificación

**Plantillas de retroalimentación**:
```
Crea plantillas de retroalimentación para:

Tipo de tarea: [ensayos/proyectos/conjuntos de problemas]
Nivel de grado: [grado]

Para cada problema común, proporciona:
- Retroalimentación constructiva
- Sugerencia específica para mejora
- Nota alentadora
- Recurso para ayuda

Haz la retroalimentación: Específica, accionable, orientada al crecimiento
```

**Comentarios de boleta de calificaciones**:
```
Escribe comentarios de boleta de calificaciones para:

Estudiante: [breve descripción de desempeño]
Materia: [materia]
Grado: [grado]

Categorías:
- Fortalezas (lo que hacen bien)
- Áreas de crecimiento (en qué trabajar)
- Notas conductuales/sociales (si aplica)
- Ítems de acción para padres (cómo apoyar en casa)

Tono: Profesional, equilibrado, constructivo
Longitud: 3-4 oraciones cada categoría
```

## Comunicación con Padres

**Correo de actualización semanal**:
```
Escribe un correo semanal para padres:

Grado: [nivel de grado]
Semana de: [rango de fechas]

Incluye:
- Qué aprendimos esta semana
- Eventos/tareas próximas
- Recordatorios de tarea
- Formas de apoyar el aprendizaje en casa
- Celebraciones/notas positivas

Tono: Amigable, informativo, alentador
Longitud: 3-4 párrafos
```

**Carta de preocupación conductual**:
```
Escribe una carta a padres sobre:

Estudiante: [solo nombre de pila]
Preocupación conductual: [problema específico]
Frecuencia: [qué tan seguido]
Impacto: [en aprendizaje/clase]

Aborda:
- Observaciones específicas (no juiciosas)
- Patrón notado
- Lo que se ha intentado en la escuela
- Solicitud de asociación
- Próximos pasos

Tono: Colaborativo, enfocado en soluciones, respetuoso
```

**Preparación de conferencia**:
```
Prepárate para conferencia de padres y maestros:

Estudiante: [perfil breve]
Temas a discutir:
- Progreso académico
- Observaciones sociales/conductuales
- Metas para resto del año

Crea:
- Puntos de conversación (positivos primero)
- Datos a compartir (calificaciones, evaluaciones, muestras de trabajo)
- Preguntas para padres
- Recomendaciones de plan de acción
- Recursos a ofrecer

Mantenlo: Equilibrado, específico, colaborativo
```

## Gestión del Aula

**Plan de gestión conductual**:
```
Crea un sistema de gestión conductual para:

Nivel de grado: [grado]
Tamaño de clase: [número]
Desafíos comunes: [conductas específicas]

Diseña:
- Expectativas claras (3-5 reglas)
- Sistema de refuerzo positivo
- Escalera de consecuencias
- Planes de estudiantes individuales
- Estrategia de comunicación con padres

Filosofía: Restaurativa, enfocada en relaciones
```

**Scripts de rutina y procedimiento**:
```
Escribe scripts para enseñar procedimientos de clase:

Procedimientos necesarios:
- [Entrar al aula]
- [Entregar trabajo]
- [Obtener ayuda]
- [Transición entre actividades]

Para cada uno:
- Por qué importa
- Proceso paso a paso
- Cómo se ve/suena
- Actividad de práctica
- Plan de refuerzo
```

## Desarrollo Curricular

**Plan de unidad**:
```
Diseña una unidad de 3 semanas sobre:

Tema: [tema]
Grado: [grado]
Materia: [materia]
Estándares: [estándares relevantes]

Incluye:
- Grandes ideas/preguntas esenciales
- Resumen de lección diaria (15 días)
- Evaluaciones formativas a lo largo
- Evaluación sumativa
- Conexiones interdisciplinarias
- Materiales/recursos necesarios
```

**Conexiones interdisciplinarias**:
```
Estoy enseñando [tema] en [materia].

Sugiere conexiones a:
- [Otra materia 1]
- [Otra materia 2]
- [Otra materia 3]

Para cada conexión:
- Idea de actividad/lección específica
- Cómo refuerza ambas materias
- Tiempo necesario
- Factor de compromiso estudiantil
```

## Apoyo de Educación Especial

**Escritura de metas IEP**:
```
Escribe metas IEP medibles para:

Estudiante: [grado, área de necesidad]
Nivel actual: [datos de línea base]
Área: [lectura/matemáticas/conducta/social]

Crea 3 metas SMART:
- Específicas
- Medibles
- Alcanzables
- Relevantes
- Con límite de tiempo

Incluye: Línea base, objetivo, método de medición, cronograma
```

**Sugerencias de acomodaciones**:
```
Estudiante con [diagnóstico/desafío]:

Lucha con: [dificultades específicas]
Fortalezas: [lo que hace bien]

Sugiere acomodaciones para:
- Entrega de instrucción
- Tareas/tarea
- Evaluaciones
- Ambiente del aula
- Apoyo social/conductual

Prioriza: Fácil de implementar, alto impacto
```

## Aprendizaje a Distancia/Híbrido

**Estructura de lección en línea**:
```
Convierte esta lección a formato en línea:

Lección original: [breve descripción]
Duración: [tiempo]
Plataforma: [Zoom/Google Meet/etc.]

Crea:
- Preparación previa a clase para estudiantes
- Actividades sincrónicas (interactivas)
- Trabajo asincrónico (independiente)
- Estrategias de compromiso para virtual
- Métodos de verificación de comprensión
- Plan de solución de problemas tecnológicos
```

**Normas de clase virtual**:
```
Crea expectativas para aprendizaje virtual:

Nivel de grado: [grado]
Plataforma: [plataforma]

Desarrolla:
- Expectativas de tecnología estudiantil
- Normas de participación
- Protocolos de cámara/audio
- Etiqueta de chat
- Procedimientos de trabajo ausente/tardío
- Aclaración de rol de padres
```

## Desarrollo Profesional

**Enfoque de observación de pares**:
```
Voy a observar a un colega enseñando [materia/grado].

Crea protocolo de observación para:

Área de enfoque: [estrategia de enseñanza específica]

Qué buscar:
- [Indicadores observables]
- [Señales de compromiso estudiantil]
- [Marcadores de calidad de implementación]

Plantilla de toma de notas:
- Observaciones objetivas
- Preguntas/curiosidades
- Ideas para traer a mi práctica
```

## Apoyo Específico de Materia

### Matemáticas
```
Crea un problema de palabras para:

Concepto: [concepto matemático específico]
Grado: [grado]
Contexto del mundo real: [escenario relatable]

Hazlo:
- Atractivo
- Claro
- Dificultad apropiada
- Incluye múltiples pasos si es apropiado para el grado
- Proporciona respuesta y solución trabajada
```

### Inglés/Artes del Lenguaje
```
Diseña un prompt de escritura para:

Grado: [grado]
Género: [narrativo/expositivo/persuasivo]
Tema: [tema general]

Incluye:
- Pregunta/escenario de prompt atractivo
- Criterios de éxito
- Organizador de pre-escritura
- Banco de palabras (si es apropiado)
- Sugerencia de texto mentor
```

### Ciencias
```
Crea un experimento práctico para:

Concepto: [concepto científico]
Grado: [grado]
Tiempo: [tiempo disponible]
Materiales: [artículos comunes/accesibles]

Incluye:
- Formación de hipótesis
- Procedimiento paso a paso
- Hoja de registro de observaciones
- Preguntas de análisis
- Conexión con el mundo real
```

### Estudios Sociales
```
Diseña una actividad de análisis de fuente primaria:

Tema: [evento/era histórica]
Grado: [grado]
Tipo de fuente: [documento/imagen/artefacto]

Crea:
- Criterios de selección de documento/imagen
- Preguntas guía
- Información de construcción de contexto
- Marco de análisis
- Conexión con hoy
```
