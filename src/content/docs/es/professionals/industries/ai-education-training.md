# IA para Educación y Capacitación

## Diseño de Cursos

```
Prompt: Diseña esquema de curso:

Tema: [Materia]
Audiencia: [Nivel de estudiante]
Duración: [Horas/semanas]
Formato: [En línea/Presencial/Híbrido]
Objetivos de aprendizaje: [Objetivos]

Crea:
- Desglose de módulos
- Objetivos de aprendizaje por módulo
- Actividades/tareas
- Métodos de evaluación
- Recursos necesarios
```

## Planificación de Lecciones

```
Prompt: Crea plan de lección:

Tema: [Lección específica]
Duración: [Minutos]
Grado/nivel: [Nivel]
Conocimiento previo: [Prerequisitos]

Incluye:
- Objetivos de aprendizaje
- Materiales necesarios
- Introducción (gancho)
- Actividades principales
- Ejercicios de práctica
- Evaluación
- Estrategias de diferenciación
```

## Creación de Evaluaciones

```
Prompt: Genera evaluación:

Tema: [Materia]
Tipo: [Examen/Prueba/Proyecto]
Preguntas: [Número]
Dificultad: [Nivel]
Formato: [Opción múltiple/Ensayo/Mixto]

Incluye:
- Tipos de preguntas variadas
- Clave de respuestas
- Rúbrica (si aplica)
- Estimación de tiempo
```

## Tutoría Personalizada

```
Prompt: Actúa como tutor para [materia]:

Nivel de estudiante: [Grado/competencia]
Estilo de aprendizaje: [Visual/Auditivo/Kinestésico]
Tema: [Tema actual]

Tutoriza:
- Evaluando comprensión
- Explicando conceptos
- Proporcionando ejemplos
- Haciendo preguntas guía
- Dando problemas de práctica
- Ofreciendo aliento
```

## Asistencia de Calificación

```
Prompt: Evalúa este ensayo:

Tarea: [Prompt]
Rúbrica: [Criterios]
Ensayo: [Trabajo del estudiante]

Proporciona:
- Puntuación por criterio
- Calificación general
- Fortalezas
- Áreas de mejora
- Retroalimentación específica
- Sugerencias para revisión
```

## Herramientas de IA para Educación

- **Khan Academy**: Tutoría con IA (Khanmigo)
- **Duolingo**: Aprendizaje adaptativo
- **Quizlet**: Conjuntos de estudio con IA
- **Gradescope**: Calificación con IA
- **Century**: Aprendizaje personalizado
