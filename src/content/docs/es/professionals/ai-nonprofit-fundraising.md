# IA para Organizaciones sin Fines de Lucro y Recaudación de Fondos

## Comunicaciones con Donantes

```
Prompt: Escribe carta de solicitud a donantes:

Campaña: [Nombre de campaña]
Objetivo: $[cantidad]
Causa: [Describe impacto]
Audiencia: [Donantes mayores/Donantes anuales]
Tono: [Sincero/Urgente/Inspirador]

Incluye:
- Historia convincente
- Impacto específico
- Solicitud clara
- Múltiples niveles de donación
- Agradecimiento
```

## Redacción de Subvenciones

```
Prompt: Redacta propuesta de subvención:

Subvención: [Nombre de fundación]
Cantidad: $[solicitado]
Proyecto: [Descripción del proyecto]
Impacto: [Resultados esperados]
Presupuesto: [Resumen]

Secciones necesarias:
- Resumen ejecutivo
- Declaración de necesidad
- Objetivos y metas
- Métodos
- Evaluación
- Justificación de presupuesto
```

## Reportes de Impacto

```
Prompt: Crea reporte de impacto:

Programa: [Nombre del programa]
Período: [Marco de tiempo]
Métricas: [Datos]
Historias: [Ejemplos de beneficiarios]

Formato para:
- Reporte anual
- Boletín de donantes
- Presentación para junta directiva

Enfatiza resultados e historias.
```

## Segmentación de Donantes

```
Prompt: Segmenta donantes:

Datos de donantes: [Estadísticas resumidas]
Patrones de donación: [Frecuencia, cantidades]
Compromiso: [Asistencia a eventos, etc]

Crea segmentos:
- Características
- Potencial de donación
- Estrategia de compromiso
- Preferencias de comunicación
```

## Contenido de Redes Sociales

```
Prompt: Crea publicaciones de redes sociales:

Campaña: [Nombre]
Plataforma: [Facebook/Instagram/LinkedIn]
Objetivo: [Conciencia/Donaciones/Reclutamiento de voluntarios]

Genera:
- 10 ideas de publicaciones
- Formatos variados (historia, estadísticas, testimonio)
- Hashtags
- Llamados a la acción
```

## Gestión de Voluntarios

```
Prompt: Crea incorporación de voluntarios:

Rol: [Posición]
Compromiso: [Horas/duración]
Habilidades necesarias: [Lista]

Incluye:
- Email de bienvenida
- Esquema de capacitación
- Expectativas
- Recursos
- Plan de reconocimiento
```
