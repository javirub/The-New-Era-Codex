# IA para Análisis Financiero

## Análisis de Estados Financieros

**Prompt de análisis de ratios**:
```
Analiza estos ratios financieros para la Empresa X:

Ratio Corriente: 1.8
Deuda a Capital: 0.45
ROE: 18%
Margen Bruto: 42%
Margen Operativo: 15%

Promedios de la industria:
Ratio Corriente: 1.5
Deuda a Capital: 0.6
ROE: 12%
Margen Bruto: 35%
Margen Operativo: 10%

Proporciona:
- Qué nos dice cada ratio
- Comparación con la industria
- Fortalezas y debilidades
- Señales de alerta o señales positivas
```

**Análisis de tendencias**:
```
Aquí está el crecimiento de ingresos de la Empresa Y durante 5 años:
Año 1: $10M
Año 2: $15M
Año 3: $18M
Año 4: $17M
Año 5: $22M

Analiza:
- Patrón de crecimiento
- Caída del Año 4 (¿qué podría explicarlo?)
- ¿Es esto sostenible?
- ¿Qué preguntas debería hacer a la gerencia?
```

## Investigación de Inversiones

**Análisis de empresa**:
```
Estoy investigando [Nombre de Empresa] como inversión potencial.

Ayúdame a analizar:
- Modelo de negocio (cómo ganan dinero)
- Ventajas competitivas/foso económico
- Riesgos clave
- Impulsores de crecimiento
- Valoración (¿es justo el precio actual?)
- Indicadores de calidad de gerencia

Proporciona marco para debida diligencia.
```

**Comparación de industria**:
```
Compara estas 3 empresas en [industria]:

Empresa A: [métricas clave]
Empresa B: [métricas clave]
Empresa C: [métricas clave]

¿Cuál es:
- ¿Mejor valor?
- ¿Mayor potencial de crecimiento?
- ¿Más segura/menor riesgo?
- ¿Mejor para ingresos por dividendos?

Explica el razonamiento para cada una.
```

## Gestión de Portafolio

**Asignación de activos**:
```
Ayúdame a construir un portafolio:

Perfil:
- Edad: 35
- Tolerancia al riesgo: Moderada
- Meta: Retiro en 25 años
- Inversión inicial: $50,000
- Contribución mensual: $1,000

Sugiere:
- Asignación de activos (acciones/bonos/alternativos)
- Diversificación geográfica
- Pesos de sectores
- Estrategia de rebalanceo
```

**Análisis de diversificación**:
```
Revisa mi portafolio para diversificación:

Tenencias:
- Acciones tecnológicas: 45%
- Acciones financieras: 20%
- Bonos: 25%
- Efectivo: 10%

Todas empresas basadas en EE.UU.

¿Cuáles son los riesgos? ¿Cómo puedo mejorar la diversificación?
```

## Análisis de Mercado

**Interpretación de tendencias**:
```
El S&P 500 ha:
- Caído 5% en el último mes
- Sector tecnológico bajó 8%
- Energía subió 3%
- VIX subió 30%

¿Qué podría indicar esto sobre:
- Sentimiento del mercado
- Preocupaciones económicas
- Rotación de sectores
- Oportunidades potenciales
```

**Análisis de indicadores económicos**:
```
Datos económicos recientes:
- Desempleo: 4.2%
- Inflación: 3.1%
- Crecimiento del PIB: 2.5%
- Tasa de fondos federales: 5.25%

¿Cómo debería esto influir:
- Asignación de acciones vs. bonos
- Preferencias de sectores
- Exposición internacional
- Duración para bonos
```

## Evaluación de Riesgo

**Perfilado de riesgo**:
```
Evalúa el riesgo de esta inversión:

Activo: [Descripción]
Volatilidad: [Número]
Correlación con el mercado: [Número]
Liquidez: [Evaluación]
Apalancamiento: [Si hay]

Comparado con mi tolerancia al riesgo de portafolio, ¿es esto apropiado?
```

**Análisis de escenarios**:
```
Si las tasas de interés suben 1% en los próximos 6 meses, ¿cómo afectaría probablemente esto:

Mi portafolio:
- 30% acciones de crecimiento
- 20% acciones de dividendos
- 40% bonos (duración promedio: 7 años)
- 10% bienes raíces

¿Qué debería hacer para prepararme?
```

## Modelos de Valoración

**Marco DCF**:
```
Ayúdame a construir un modelo DCF simple para [Empresa]:

Entradas:
- Flujo de caja libre actual: $500M
- Tasa de crecimiento esperada: 8% por 5 años, luego 3%
- Tasa de descuento (WACC): 10%

Guíame a través de:
1. Proyección de flujos de caja
2. Cálculo de valor terminal
3. Descuento a valor presente
4. Interpretación del resultado
```

**Análisis de comparables**:
```
La Empresa X cotiza a:
- P/E: 25
- P/S: 4
- EV/EBITDA: 18

Empresas similares promedian:
- P/E: 20
- P/S: 3
- EV/EBITDA: 15

¿Está la Empresa X sobrevaluada? ¿Qué factores podrían justificar la prima?
```

## Análisis de Ganancias

**Resumen de llamada de ganancias**:
```
Resume esta transcripción de llamada de ganancias:

[pega transcripción o citas clave]

Extrae:
- Cambios en orientación
- Nuevas iniciativas
- Preocupaciones de gerencia
- Amenazas competitivas
- Oportunidades de crecimiento
- Señales de alerta en lenguaje/tono
```

**Impacto de sorpresa de ganancias**:
```
La empresa reportó:
- EPS: $2.50 (esperado $2.30)
- Ingresos: $5B (esperado $4.8B)
- Orientación elevada en 10%

La acción cayó 5% después de horas.

¿Por qué podría pasar esto? ¿Qué me estoy perdiendo?
```

## Apoyo de Análisis Técnico

**Interpretación de patrón de gráfico**:
```
La Acción X ha formado [describe patrón - cabeza y hombros, doble fondo, etc.]

Durante [marco de tiempo]

¿Qué sugiere típicamente este patrón?
¿Qué debería observar para confirmación?
¿Cuál es el objetivo/stop loss?
¿Qué tan confiable es este patrón?
```

**Análisis de indicador**:
```
La Acción Y muestra:
- RSI: 72
- MACD: Cruce alcista
- Media móvil de 50 días: Acción cotizando arriba
- Volumen: Arriba del promedio

¿Cuál es la señal general? ¿Cuáles son las contradicciones?
```

## Inversión en Bienes Raíces

**Análisis de propiedad**:
```
Evalúa esta propiedad de alquiler:

Precio de compra: $300,000
Ingreso de alquiler: $2,000/mes
Gastos: $600/mes (impuestos, seguro, mantenimiento)
Pago inicial: 20%
Tasa hipotecaria: 6.5%
Ubicación: [Área en crecimiento/estable/declive]

Calcula:
- Tasa de capitalización
- Retorno cash-on-cash
- Análisis de punto de equilibrio
- ¿Es esta una buena inversión?
```

## Optimización Fiscal

**Cosecha de pérdidas fiscales**:
```
Tengo:
- Acción A: Abajo $5,000 (aún creo en largo plazo)
- Acción B: Arriba $8,000 (quiero tomar ganancias)

Ayúdame a planear:
- Estrategia de cosecha de pérdidas fiscales
- Consideraciones de regla de venta de lavado
- Momento óptimo
- Valores de reemplazo
```

**Optimización de cuenta**:
```
Tengo: 401(k), Roth IRA, Cuenta de corretaje tributable

Inversiones a asignar:
- Acciones de alto dividendo
- Acciones de crecimiento (bajos dividendos)
- Bonos
- REITs

¿Qué cuentas deberían tener qué activos para eficiencia fiscal?
```

## Planificación de Retiro

**Cálculo de necesidades**:
```
Planificación de retiro:

Edad actual: 40
Edad de retiro: 65
Salario actual: $100,000
Gastos esperados en retiro: 80% del actual
Expectativa de vida: 90

¿Cuánto necesito ahorrar?
¿Estoy en camino con: $250,000 ahorrados actualmente, $1,500/mes de contribuciones?
```

**Estrategia de retiro**:
```
Me estoy retirando con:
- 401(k): $800,000
- Roth IRA: $200,000
- Cuenta tributable: $300,000
- Seguro Social: $2,500/mes (comenzando a los 67)

Necesito $6,000/mes.

Crea una estrategia de retiro fiscalmente eficiente para edades 62-70.
```

## Interpretación de Noticias de Mercado

**Análisis de impacto de noticias**:
```
Noticias: "La Reserva Federal señala posibles recortes de tasas en 2024"

¿Cómo podría esto afectar:
- Mercado de valores (¿qué sectores se benefician/perjudican?)
- Precios de bonos
- Bienes raíces
- Fortaleza del dólar
- Mercados emergentes

¿Debería ajustar mi portafolio?
```
