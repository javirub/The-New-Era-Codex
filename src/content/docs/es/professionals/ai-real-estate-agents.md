# IA para Agentes Inmobiliarios

## Listados de Propiedades

**Descripción de listado**:
```
Crea una descripción de listado convincente para:

Tipo de propiedad: [Casa unifamiliar / Condominio / Multifamiliar]
Habitaciones: [número]
Baños: [número]
Pies cuadrados: [tamaño]
Tamaño de lote: [si aplica]
Año de construcción: [año]
Precio: $[precio]
Ubicación: [vecindario/ciudad]

Características clave:
- [Cocina renovada]
- [Pisos de madera]
- [Patio trasero]
- [Características únicas]

Cercano:
- [Escuelas]
- [Compras]
- [Parques]
- [Transporte]

Tono: [Lujo / Familiar / Inversión / Comprador primerizo]
Longitud: 150-250 palabras

Incluye: Apertura que capte atención, detalles sensoriales, beneficios de estilo de vida, llamado a la acción
```

**Publicaciones de redes sociales para listados**:
```
Crea 5 publicaciones de redes sociales para este listado:

Propiedad: [breve descripción]
Audiencia objetivo: [tipo de comprador]
Plataformas: Instagram, Facebook

Para cada publicación:
- Gancho/texto (apertura atractiva)
- Destacado de características clave
- Llamado a la acción
- Hashtags relevantes

Formatos:
1. Anuncio de recién listado
2. Destacado de característica (cocina/patio/vista)
3. Promoción de casa abierta
4. Reducción de precio (si aplica)
5. Ángulo de testimonio de comprador
```

**Script de recorrido virtual**:
```
Escribe un script de video-recorrido para:

Propiedad: [descripción]
Puntos de venta únicos: [lista]
Duración: 3-5 minutos

Estructura:
- Apertura (atractivo exterior, exterior)
- Áreas de estar principales (flujo y características)
- Cocina (detalle en actualizaciones/características)
- Habitaciones (destacar principal)
- Espacio exterior
- Beneficios del vecindario
- Cierre (llamado a la acción)

Tono: Conversacional, entusiasta, informativo
```

## Comunicación Comprador/Vendedor

**Consulta inicial de comprador**:
```
Crea un cuestionario de consulta de comprador:

Temas a cubrir:
- Imprescindibles vs. deseables
- Presupuesto y estado de financiamiento
- Cronograma
- Preferencias de vecindario
- Necesidades de estilo de vida (viaje, escuelas, etc.)
- Rompedores de trato

Formato como:
- Guía de conversación con preguntas de seguimiento
- Plantilla de toma de notas
- Bosquejo de próximos pasos
```

**Reporte de análisis de mercado**:
```
Genera una narrativa de CMA (Análisis de Mercado Comparativo) para:

Propiedad sujeto: [dirección, especificaciones básicas]
Ventas comparables:
1. [Dirección, precio, especificaciones, DOM]
2. [Dirección, precio, especificaciones, DOM]
3. [Dirección, precio, especificaciones, DOM]

Condiciones de mercado: [caliente/equilibrado/lento]

Crea reporte incluyendo:
- Resumen de mercado
- Análisis de comparables
- Recomendación de precio con justificación
- Estrategia de marketing
- Expectativas de cronograma

Para: [Presentación a vendedor / Cita de listado]
```

**Presentación de oferta**:
```
Presenta esta oferta a mi vendedor:

Precio de oferta: $[cantidad]
Dinero en serio: $[cantidad]
Financiamiento: [Efectivo / Convencional / FVA / etc.]
Contingencias: [Inspección / Financiamiento / Avalúo / Venta de casa del comprador]
Fecha de cierre: [fecha]
Otros términos: [Concesiones del vendedor / Artículos incluidos / Alquiler posterior]

Precio de listado: $[cantidad]
DOM: [días en mercado]
Otros factores: [¿Múltiples ofertas? / ¿Preocupaciones de avalúo?]

Crea presentación delineando:
- Resumen de oferta
- Indicadores de fortaleza del comprador
- Pros y contras
- Estrategia de negociación
- Recomendación
```

## Generación de Leads

**Campaña de correo por goteo**:
```
Crea una secuencia de 6 correos de nutrición para:

Audiencia: [Compradores primerizos / Vendedores considerando / Inversionistas]
Meta: [Construir relación, educar, programar consulta]

Cronograma de correo: Semana 1, Semana 2, Semana 4, Semana 6, Semana 8, Semana 10

Para cada correo:
- Línea de asunto (alta tasa de apertura)
- Contenido orientado a valor (consejos, ideas de mercado, recursos)
- Llamado a la acción suave
- Construcción de relación progresiva

Tono: Experto útil, no ventas insistentes
```

**Seguimiento de casa abierta**:
```
Escribe mensajes de seguimiento para visitantes de casa abierta:

Segmentos:
1. Interés serio (hizo preguntas detalladas)
2. Observador casual (solo navegando)
3. Vecino (curioso sobre el valor)

Para cada uno:
- Línea de asunto de correo
- Cuerpo del mensaje
- Preguntas específicas para calificar interés
- Sugerencia de próximo paso
- Cronograma para seguimiento

Puntos de personalización a incluir.
```

**Divulgación de esfera de influencia**:
```
Crea boletín mensual para mi base de datos:

Tema de este mes: [Mercado de primavera / Días festivos / Regreso a clases / etc.]

Incluye secciones sobre:
- Actualización de mercado local (estadísticas breves)
- Listado destacado
- Consejo de mantenimiento del hogar para la temporada
- Destacado de comunidad
- Historia de éxito de cliente o testimonio
- Nota/actualización personal

Longitud: 400-600 palabras
Tono: Personal, vecinal, informativo
```

## Educación del Cliente

**Guía de comprador primerizo**:
```
Crea una guía "Qué Esperar" para compradores primerizos:

Cubrir:
- Cronograma de pre-aprobación a cierre
- Explicación de proceso paso a paso
- Términos y definiciones clave
- Qué traer al cierre
- Costos a esperar (más allá del pago inicial)
- Errores comunes a evitar
- Preguntas a hacer durante visitas

Formato: Folleto PDF o serie de correo
Nivel de lectura: Accesible, no demasiado técnico
```

**Lista de verificación de venta de casa**:
```
Crea una lista de verificación pre-listado para vendedores:

Marco de tiempo: 4-6 semanas antes de listar

Semana 1-2: [Despejar, reparaciones, limpieza profunda]
Semana 3-4: [Staging, retoques, atractivo exterior]
Semana 5-6: [Fotos profesionales, preparación final, listado]

Para cada ítem:
- Tarea específica
- Por qué importa
- Hacerlo tú mismo vs. contratar
- Costo estimado
- Impacto en precio/velocidad de venta

Prioridad: Alta / Media / Baja para cada tarea
```

**Análisis de propiedad de inversión**:
```
Crea un análisis de propiedad de inversión para cliente:

Propiedad: [dirección/detalles]
Precio de compra: $[cantidad]
Renta esperada: $[mensual]
Gastos estimados: [impuestos, seguro, mantenimiento, HOA]
Pago inicial: [porcentaje]
Términos de hipoteca: [tasa, plazo]

Calcula y presenta:
- Flujo de caja mensual
- Tasa de capitalización
- Retorno cash-on-cash
- Proyecciones de ROI (1 año, 5 años, 10 años)
- Supuestos de apreciación
- Resumen de beneficios fiscales

Formato: Resumen de una página con notas
```

## Negociación y Contratos

**Estrategia de contraoferta**:
```
Ayúdame a elaborar una respuesta de contraoferta:

Oferta original: $[cantidad]
Nuestro precio de listado: $[cantidad]
Nuestro precio mínimo: $[cantidad]
Apalancamiento de mercado: [Mercado de vendedor / Mercado de comprador / Equilibrado]

Problemas más allá del precio:
- [Preocupaciones de fecha de cierre]
- [Solicitudes de contingencia]
- [Negociaciones de reparación]

Crea:
- Estrategia de contraoferta (qué pedir, qué conceder)
- Justificación para nuestra posición
- Soluciones alternativas
- Qué comunicar a la otra parte
```

**Negociación de inspección**:
```
La inspección del comprador encontró:

Problemas mayores: [Lista con costos estimados de reparación]
Problemas menores: [Lista]

Perspectiva del cliente: [Comprador quiere créditos / Vendedor no quiere pagar]

Crea enfoque de negociación:
- Qué es razonable solicitar
- Cómo priorizar ítems
- Soluciones creativas (créditos vs. reparaciones vs. reducción de precio)
- Umbral de rompedor de trato
- Propuestas de compromiso
```

## Marketing en Redes Sociales

**Calendario de contenido de Instagram**:
```
Crea un plan de contenido de Instagram de 30 días:

Mezcla:
- 40% Listados (míos y otros para mostrar conocimiento de mercado)
- 30% Contenido de comunidad/estilo de vida local
- 20% Educativo (consejos, ideas de mercado)
- 10% Personal/detrás de escenas

Para cada semana, especifica:
- 5-7 ideas de publicación
- Marcos de texto
- Mejores horarios de publicación
- Ideas de historia para complementar
- Estrategias de compromiso (encuestas, preguntas, etc.)

Estrategia de hashtag para mi mercado: [Ciudad/vecindario]
```

**Anuncios de Facebook para listados**:
```
Crea copia de anuncio de Facebook para:

Listado: [descripción de propiedad]
Audiencia objetivo: [demografía, radio de ubicación]
Presupuesto: $[diario/total]
Meta de campaña: [Leads / Asistencia a casa abierta / Conciencia de marca]

Proporciona:
- 3 variaciones de titular
- Texto principal (125 caracteres)
- Descripción
- Elección de botón de llamado a la acción
- Sugerencias de targeting de audiencia
- Orientación de selección de imagen
```

## Gestión de Tiempo y Productividad

**Plantilla de horario diario**:
```
Crea un horario diario óptimo para:

Prioridades:
- Actividades de generación de leads
- Citas con clientes
- Administración/papeleo
- Marketing
- Desarrollo profesional

Horas disponibles: [horas de trabajo]
Patrones de energía: [Persona mañanera / Noctámbulo]

Horario en bloques incluyendo:
- Tiempo de enfoque para actividades de alto valor
- Tiempo de cara al cliente
- Tiempo de buffer
- Procesamiento por lotes (llamadas, correos, redes sociales)
- Bloques no negociables
```

**Lista de verificación de transacción**:
```
Crea una lista de verificación de gestión de transacción de contrato a cierre:

Para transacciones de comprador:
- [Cronograma de aceptación de oferta a cierre]
- [Hitos y fechas límite clave]
- [Documentos necesarios]
- [Coordinación con prestamista, título, inspector]
- [Puntos de contacto de comunicación con cliente]

Para transacciones de vendedor:
- [Hitos relevantes diferentes]

Incluye: Quién es responsable, fecha límite, seguimiento de estado
```

## Experiencia de Mercado

**Guía de vecindario**:
```
Crea una guía de vecindario para:

Vecindario: [nombre]
Compradores objetivo: [familias / jóvenes profesionales / jubilados / etc.]

Incluye:
- Resumen y vibra
- Inventario de vivienda y rangos de precio
- Escuelas (calificaciones y destacados)
- Parques y recreación
- Restaurantes y compras
- Transporte y viaje
- Desarrollo futuro
- Por qué a la gente le encanta vivir aquí
- Consejos de insider

Formato: Publicación de blog o descarga PDF
Uso: Imán de leads, recurso de cliente, contenido SEO
```

**Script de video de actualización de mercado**:
```
Escribe un script de actualización de mercado mensual para:

Mercado: [Ciudad/vecindario]
Datos de este mes:
- Precio de venta mediano: $[cantidad] ([arriba/abajo X%] del mes pasado)
- Días en mercado: [número]
- Inventario: [número de listados activos]
- Ratio de listado a venta: [porcentaje]

Crea script de 2-3 minutos cubriendo:
- Gancho (estadística o tendencia interesante)
- Números clave (explicados para no expertos)
- Qué significa esto para compradores
- Qué significa esto para vendedores
- Predicción/perspectiva
- Llamado a la acción

Tono: Autoritativo pero accesible
```

## Cultivo y Prospección

**Script de tocar puertas**:
```
Crea un script de tocar puertas para:

Propósito: [Recién listado / Recién vendido / Chequeo de mercado]
Vecindario: [área objetivo]

El script debería incluir:
- Introducción (quién eres)
- Razón de visita (específica, relevante)
- Oferta de valor (reporte de mercado, valoración de casa, etc.)
- Respuestas a objeciones
- Próximo paso/seguimiento
- Salida con gracia

Tono: Amigable, profesional, no insistente
Variaciones para: Sin respuesta, no interesado, dispuesto a charlar
```

**Carta de listado vencido**:
```
Escribe una carta al propietario de listado vencido:

Propiedad: [si se conoce, o general]
Días en mercado: [si se conoce]
Puntos de precio: [si se conoce]

La carta debería:
- Reconocer frustración (empatía)
- Identificar razones comunes por las que los listados vencen
- Destacar tu enfoque diferente
- Proporcionar prueba social (ventas recientes, testimonios)
- Ofrecer consulta gratuita y análisis de mercado
- Llamado a la acción claro

Tono: Comprensivo, confiado, enfocado en soluciones
Sin hablar negativamente del agente anterior
```
