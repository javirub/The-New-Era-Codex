# AI for Nonprofits & Fundraising

## Donor Communications

```
Prompt: Write donor appeal letter:

Campaign: [Campaign name]
Goal: $[amount]
Cause: [Describe impact]
Audience: [Major donors/Annual givers]
Tone: [Heartfelt/Urgent/Inspirational]

Include:
- Compelling story
- Specific impact
- Clear ask
- Multiple giving levels
- Thank you
```

## Grant Writing

```
Prompt: Draft grant proposal:

Grant: [Foundation name]
Amount: $[requested]
Project: [Project description]
Impact: [Expected outcomes]
Budget: [Summary]

Sections needed:
- Executive summary
- Need statement
- Goals and objectives
- Methods
- Evaluation
- Budget justification
```

## Impact Reporting

```
Prompt: Create impact report:

Program: [Program name]
Period: [Timeframe]
Metrics: [Data]
Stories: [Beneficiary examples]

Format for:
- Annual report
- Donor newsletter
- Board presentation

Emphasize outcomes and stories.
```

## Donor Segmentation

```
Prompt: Segment donors:

Donor data: [Summary stats]
Giving patterns: [Frequency, amounts]
Engagement: [Event attendance, etc]

Create segments:
- Characteristics
- Giving potential
- Engagement strategy
- Communication preferences
```

## Social Media Content

```
Prompt: Create social media posts:

Campaign: [Name]
Platform: [Facebook/Instagram/LinkedIn]
Goal: [Awareness/Donations/Volunteer recruitment]

Generate:
- 10 post ideas
- Varied formats (story, stats, testimonial)
- Hashtags
- Calls-to-action
```

## Volunteer Management

```
Prompt: Create volunteer onboarding:

Role: [Position]
Commitment: [Hours/duration]
Skills needed: [List]

Include:
- Welcome email
- Training outline
- Expectations
- Resources
- Recognition plan
```
