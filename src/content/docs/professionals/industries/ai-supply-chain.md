# AI for Supply Chain & Operations

## Demand Forecasting

```
Prompt: Forecast demand:

Product: [Product name]
Historical data: [Sales data]
Seasonality: [Patterns]
External factors: [Events/trends]
Timeline: [Next 3/6/12 months]

Provide:
- Demand forecast
- Confidence intervals
- Key assumptions
- Risk factors
- Inventory recommendations
```

## Inventory Optimization

```
Prompt: Optimize inventory:

Current: [Stock levels]
Lead times: [Days]
Demand: [Forecast]
Storage cost: $[amount]
Stockout cost: $[amount]

Recommend:
- Reorder points
- Safety stock levels
- Order quantities
- Cost optimization
```

## Supplier Analysis

```
Prompt: Evaluate suppliers:

Suppliers: [List with performance data]
Criteria:
- Quality
- Price
- Delivery time
- Reliability
- Risk factors

Rank and recommend with reasoning.
```

## Route Optimization

```
Prompt: Optimize delivery routes:

Locations: [Addresses]
Constraints:
- Vehicle capacity
- Time windows
- Driver hours
- Priority deliveries

Suggest optimal routes with time/cost savings.
```
