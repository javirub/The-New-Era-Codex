# AI for Real Estate

## Property Descriptions

```
Prompt: Write property listing:

Property details:
- Type: [House/Condo/etc]
- Beds/Baths: [Numbers]
- Sq ft: [Size]
- Location: [Neighborhood]
- Features: [List highlights]
- Price: $[amount]

Target: [First-time buyers/Investors/Luxury]
Tone: [Professional/Warm/Sophisticated]

Include:
- Compelling headline
- Key features
- Neighborhood highlights
- Call-to-action
```

## Market Analysis

```
Prompt: Analyze real estate market:

Location: [Area]
Property type: [Type]
Price range: $[range]
Time period: [Last X months]

Data: [Market data]

Provide:
- Market trends
- Price movements
- Inventory levels
- Days on market
- Buyer/seller market analysis
- Forecast
```

## Investment Analysis

```
Prompt: Evaluate investment property:

Property: [Address/details]
Purchase price: $[amount]
Rent potential: $[monthly]
Expenses: [List]
Down payment: [%]
Loan terms: [Details]

Calculate:
- Cash flow
- CAP rate
- ROI
- Break-even analysis
- Risks
- Recommendation
```

## Email Templates

```
Prompt: Write follow-up email:

Recipient: [Buyer/Seller]
Context: [Showing/Offer/etc]
Property: [Address]
Next steps: [What you want]

Tone: Professional yet personable
```

## AI Real Estate Tools
