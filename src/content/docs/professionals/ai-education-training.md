# AI for Education & Training

## Course Design

```
Prompt: Design course outline:

Topic: [Subject]
Audience: [Student level]
Duration: [Hours/weeks]
Format: [Online/In-person/Hybrid]
Learning goals: [Objectives]

Create:
- Module breakdown
- Learning objectives per module
- Activities/assignments
- Assessment methods
- Resources needed
```

## Lesson Planning

```
Prompt: Create lesson plan:

Topic: [Specific lesson]
Duration: [Minutes]
Grade/level: [Level]
Prior knowledge: [Prerequisites]

Include:
- Learning objectives
- Materials needed
- Introduction (hook)
- Main activities
- Practice exercises
- Assessment
- Differentiation strategies
```

## Assessment Creation

```
Prompt: Generate assessment:

Topic: [Subject matter]
Type: [Quiz/Test/Project]
Questions: [Number]
Difficulty: [Level]
Format: [Multiple choice/Essay/Mixed]

Include:
- Varied question types
- Answer key
- Rubric (if applicable)
- Time estimate
```

## Personalized Tutoring

```
Prompt: Act as tutor for [subject]:

Student level: [Grade/proficiency]
Learning style: [Visual/Auditory/Kinesthetic]
Topic: [Current topic]

Tutor by:
- Assessing understanding
- Explaining concepts
- Providing examples
- Asking guiding questions
- Giving practice problems
- Offering encouragement
```

## Grading Assistance

```
Prompt: Evaluate this essay:

Assignment: [Prompt]
Rubric: [Criteria]
Essay: [Student work]

Provide:
- Score per criterion
- Overall grade
- Strengths
- Areas for improvement
- Specific feedback
- Suggestions for revision
```
