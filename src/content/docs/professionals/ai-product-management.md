# AI for Product Management

## Product Strategy

```
Prompt: Analyze product strategy:

Product: [Name]
Market: [Target market]
Competitors: [List]
Current state: [Description]
Goals: [Objectives]

Provide:
- Market positioning analysis
- Differentiation opportunities
- Strategic recommendations
- Risks
- Success metrics
```

## User Research Analysis

```
Prompt: Analyze user feedback:

Feedback: [Interview notes/survey responses]

Extract:
- Common pain points
- Feature requests
- User personas patterns
- Prioritized themes
- Actionable insights
- Quote highlights
```

## Feature Prioritization

```
Prompt: Help prioritize features:

Features: [List with descriptions]

Criteria:
- User impact (1-10)
- Business value (1-10)
- Implementation effort (1-10)
- Strategic alignment (1-10)

Rank using RICE or similar framework.
Explain reasoning.
```

## Product Requirements

```
Prompt: Write PRD for:

Feature: [Feature name]
Problem: [User problem]
Goals: [What it achieves]
Users: [Target users]
Success metrics: [KPIs]

Include:
- User stories
- Acceptance criteria
- Edge cases
- Dependencies
- Open questions
```

## Roadmap Planning

```
Prompt: Create product roadmap:

Timeline: [Next 6-12 months]
Themes: [Strategic themes]
Resources: [Team size]
Constraints: [Technical/business]

Format:
- Now/Next/Later
- Key initiatives per quarter
- Dependencies
- Success metrics
```

## Competitive Analysis

```
Prompt: Competitive analysis:

Our product: [Description]
Competitors: [List]
Features to compare: [List]

Create:
- Feature comparison matrix
- Strengths/weaknesses
- Market gaps
- Opportunities
- Threat assessment
```

## Release Communications

```
Prompt: Write release notes:

Version: [Number]
Audience: [Users/Technical]
Changes: [List]

Format:
- What's new
- Improvements
- Bug fixes
- Known issues
- Migration guide (if needed)

Tone: [Clear, exciting, helpful]
```
