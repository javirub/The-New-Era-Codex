# AI for Legal Research & Analysis

## Contract Review

**Key terms extraction**:
```
Review this contract and extract:

[paste contract section or full contract]

Identify:
- Key obligations for each party
- Payment terms
- Termination clauses
- Liability limitations
- Unusual or concerning provisions
- Missing standard protections
```

**Risk assessment**:
```
Analyze this vendor contract for risks:

[paste contract]

Focus on:
- Financial exposure
- Liability caps (adequate or too low?)
- Indemnification (one-sided?)
- Auto-renewal clauses
- Data privacy provisions
- Dispute resolution (favorable venue?)

Rate each risk: High/Medium/Low
```

**Redlining suggestions**:
```
I need to negotiate this contract clause:

Current language:
"[paste clause]"

Issues with current language:
- [your concerns]

Suggest:
- Alternative language that protects my interests
- Industry-standard wording
- What to push back on vs. accept
```

## Legal Research

**Case law research**:
```
I need precedents for:

Issue: [legal question]
Jurisdiction: [state/federal]
Practice area: [area of law]

Help me:
- Frame the legal question
- Identify relevant case law
- Summarize key holdings
- Distinguish unfavorable cases
- Find supporting arguments
```

**Statutory interpretation**:
```
Interpret this statute:

[paste statute text]

Context: [situation you're applying it to]

Explain:
- Plain language meaning
- Key terms definitions
- Exceptions or carve-outs
- How it applies to my situation
- Potential ambiguities
```

## Legal Writing

**Demand letter**:
```
Draft a demand letter for:

Situation: [describe dispute]
Recipient: [party]
What you want: [remedy]
Tone: [Firm but professional / threatening litigation / settlement-oriented]

Include:
- Statement of facts
- Legal basis for claim
- Demand/remedy
- Deadline for response
- Consequences if ignored
```

**Motion to dismiss draft**:
```
Draft arguments for motion to dismiss based on:

Grounds: [lack of jurisdiction / failure to state a claim / statute of limitations]
Facts: [relevant facts]
Applicable law: [jurisdiction]

Structure: Introduction, Standard of Review, Argument, Conclusion
```

**M&A checklist**:
```

Target: [type of company]

Categories:
- Corporate structure
- Contracts and agreements
- Intellectual property
- Litigation and compliance
- Employment matters

```

**Compliance audit**:
```

Create an audit checklist:
- What we need to review
- Documents to gather
- Processes to examine
- Common violations to check for
- Remediation steps if gaps found
```

**NDA template**:
```

Duration: 2 years
Jurisdiction: [state]

Include:
- Definition of confidential information
- Permitted uses
- Exclusions from confidentiality
- Return/destruction of materials
- Remedies for breach
```

**Service agreement outline**:
```
Outline a service agreement for:

Service provider: [type]

Services: [description]
Payment: [structure]

Key sections needed:
- Scope of work
- Deliverables and timeline
- Payment terms
- IP ownership

- Termination rights
- Liability limitations
```

## Dispute Resolution

**Negotiation strategy**:
```
I'm negotiating settlement of:

Dispute: [description]
Our position: [claim/defense]
Their position: [opposing claim]
Our best case outcome: [ideal result]
Our BATNA: [alternative if no settlement]

Create:
- Opening position
- Fallback positions
- Concession strategy
- Deal breakers
- Creative solutions to explore
```

**Mediation preparation**:
```
Prepare me for mediation:

Dispute: [issue]
Our strengths: [list]
Our weaknesses: [list]
Other party's likely arguments: [list]

Create:
- Opening statement outline
- Key points to emphasize
- Anticipated challenges and responses
- Settlement authority parameters
```

## Intellectual Property

**Trademark search strategy**:
```
I want to trademark "[proposed mark]" for [goods/services].

Help me:
- Assess likelihood of registration
- Search strategy for confusingly similar marks
- Identify potential conflicts
- Suggest variations if conflicts exist
- Understand scope of protection
```

**Copyright analysis**:
```
Is this fair use?

Use: [how you want to use copyrighted material]
Original work: [description]
Purpose: [commercial/educational/commentary]
Amount used: [portion]
Effect on market: [impact]

Analyze under fair use factors. Risk level?
```

**Patent prior art search**:
```
Invention: [description]
Key features: [novel elements]

Help me:
- Frame search query
- Identify relevant patent classes
- Key terms to search
- How to evaluate results
- What makes prior art relevant
```

```
- Establish facts
- Lock in their story
- Identify witnesses/documents
- Reveal weaknesses
```

**Deposition outline**:
```

What I need to establish: [goals]

Create deposition outline:
- Background questions
- Key topic areas
- Impeachment opportunities
- Document review questions
- Preservation of testimony for trial
```
